常用的代理工具：
1、无界、花园、自由门这些就不多说了，除了默认使用 IE 和有时候找不到代理服务器之外，还是挺方便的。
设置 Firefox/Opera 等浏览器的代理地址为本机就可以关掉 IE 使用了。
2、beHidden.com 挺容易使用，填写网址回车即可，速度时快时慢。免费用户有时候要在提醒注册的页面上选择“Not Now”才能继续浏览页面。但无法正常登录 WordPress 后台，所以通常只用来阅读页面。
3、CyberSyndrome 是一个代理服务器搜索网站，通常都找得到速度快的代理，就是操作起来麻烦一点，结合 MyIE/Maxthon、Firefox+SwitchProxy Tool 使用更好。
4、Proxy Source

缺点也是无法正常登录 WordPress 后台。
5、QQ 代理公布器，Google 一把就可以搜到，以前挺好用的，后来搜索功能失效，到现在很久没用了。
