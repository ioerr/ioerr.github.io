Don't Panic
15 Aug 2007, 17:05

主流
邮件讨论

经过了几年的实践学习，我已经深刻地领悟到，作为国内的互联网用户，最重要的原则是什么，那就是一定要靠近主流，只有主流才能够生存，这绝对是不二法门。

比如说，自从 NN 没落以来，浏览器市场就是以 IE 为主流的，如果你不幸生在了中国，并且悍然使用了非 IE 的浏览器，例如 Firefox、Opera 等等，那么在门户、网站、论坛、Blog 乃至办公网络等诸多地方就活该无法正常浏览页面，无法正常使用功能，一般这些站点上做网页开发的人员会非常认为非 IE 用户最好统统都死掉，就好像快餐店恨不得所有顾客都点标准套餐，那些变着法子专点5分钟之内没法现做的套餐的活该饿死一样。
比如说，阿里巴巴宣布支付宝不再支持 Firefox 等非 IE 内核浏览器，说了也就说了，遭到抗议也就弄篇回应对付一下，到时候不支持还是不支持，你能够如何？几家较大的网络银行就是不支持非 IE 内核浏览器，你又能够怎么样？有本事自己开银行开网站去啊～
所以再怎么样，至少也要用 IE 内核各种外壳的浏览器，千万不要站错了队伍自绝于人民，到时候没有人可怜你的。

比如说，主流的文档是 Microsoft Word 的最新版本文档，从来都不是纯文本，不是 HTML，也不是其他什么格式。如果你既不肯花钱买个正版，又不肯使用盗版，却要和其他人交流、共同编辑文档，那就等着被憋死吧。
当然，我们还有 WPS Office，但自从进入 Windows 时代以来，它自身还有什么竞争力？
至于在线合作编辑文档，那都是没有任何意义的，因为你交流文档的方式通常都是在邮件的附件中获得 Word 文档，修改后再发回去。如果你不是这样的，小心了！你已经在主流的圈子之外了。

据说，以下做法可有效打击对方士气至崩溃：
用国内各大门户的邮箱给对方的国外邮箱发邮件，标题乱码，正文为空，附件是 WinRAR 多卷多层嵌套压缩，文件名亦为乱码，密码写在压缩包注释中，压缩包内容是最新版本 Word 文档，文字部分所使用的格式和特效无所不用其极地华丽，当然低版本 Word 并不支持，其实最重要的不过是一系列桌面 BMP 格式的截图，而截图的内容是在写字板上的一句话。

比如说，我国的主流是和谐的，所以在网上千万不要接触和发布不和谐的内容，跟九个指头的手势不一致，就要当心被斩。
Blogspot 总是上不去？那玩意儿只是在米帝受欢迎，我们应该和巴基斯坦、印度进行国际接轨，在国内要向各大门户的 blog 积极靠拢嘛；
维基百科上不去？不容于主流的东西太多了嘛，完全不符合我国国情，国内有现成的百科全书，干嘛要辛辛苦苦地抄后路呢？
还有，在重要问题上，一定要紧跟主流的声音，非主流的观点不要说，非主流的关键字不要提，非主流的立场不要坚持，否则如果群众觉得你这小子屁股摆放位置朝向与众不同，自发组织就把你和谐了，连警察蜀黍都不好意思出手。

所以，当 MSN Messenger 像 QQ 一样越来越臃肿，当 TM 的界面风格不断向 MSN Messenger 靠拢，我们应该而且能够做的，就是衷心拥护这种天下和谐一家的主流。
