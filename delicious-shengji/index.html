Del.icio.us 升级
Table of contents

昨天 Del.icio.us 贴出了停机升级公告，今天恢复后，原来的一些可选的试验特性被加入默认功能中：
Inbox
Tag Bundles
Daily blog posting

Inbox 功能似乎是用来订阅其它用户的 Del.icio.us 收藏的，还可以自定义分类。但是好像还存在不少问题，有的时候明明订阅了，在 Inbox 中却不显示，明明取消订阅了，在 Inbox 中却一直存在。

Tag Bundles 是用来集中几个 Tag 的，就好像在 Tag 这一层之上又增加了一层分类。
同样的，在编辑 Bundles 的时候，之前设置的都消失了，查看的时候又会出现。

Daily blog posting 要编辑的东西太多，没有导入导出的功能，没有 RSS Feed，也没有和用户的 Del.icio.us 相结合的迹象，还不明白提供这个功能的目的是什么。
新建项目在有的页面会消失的问题照旧，但愿尽快脱离试验状态吧。

另外还发现可以按照字母顺序、使用频率和自定义的 Tag Bundle 来对 Tag 进行排序，以及查看 Tag Cloud。前一阵子没有仔细观察过，可能推出来有些时间了。
