Don't Panic
01 Aug 2005, 16:04

Book Crossing
Table of contents

呃，比较火星的内容了，不过可能还有不少人不知道，所以多嘴一下：）

如果你捡到一本找不到主人的书，你会怎么处置呢？阅读，丢掉，还是读完再丢掉？有没有想过这本书也许是它的原主人故意留给别人并要下一个人继续传播呢？

Book Crossing 就把这样一个好玩的点子变成了行动，虽然听上去更像《读者文摘》里面老掉牙的小资故事：
找一本你读完了的书；
在 Book Crossing 注册，得到一个唯一的 BCID 号码，然后制作成标签贴在这本书上；
最后，把它交给其他人，比如你的朋友，比如偷偷放在公园、咖啡店这样的公共场所，或者做善事捐出去，然后等着下一位读者通过标签找到 Book Crossing 后，让大家知道这本书旅行到哪里了。如果你打算留给陌生人，最好是在注册后，做好 Release Notes，以便这个陌生人也能通过 Active Releases List 找到这本书

如果你觉得这件事情很有趣或者很有意义，不妨试一试，也许能够通过这个方式交到不少同样爱书的朋友：）当然，把心爱的书弄丢了，也只能够认命了。
你可以自己制作一款中文的标签上传到 Book Crossing，这样就不用担心其他人因为不懂外语而错过交流和再次传播的机会。

我看了一下传播次数最多的书籍，Douglas Adams 的《The Hitchhiker’s Guide to the Galaxy》以22次排在第79位。

顺便说一句，Book Crossing 在2005年的第9界 Webby Awards 中获得 People’s Voice Awards 中的 Best Community Website 与 Best Social Networking Website 奖项。虽然这个网站很早就出现了…
