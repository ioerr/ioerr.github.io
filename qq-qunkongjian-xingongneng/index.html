QQ 群空间新功能
Table of contents

今天通过 Datou 知道腾讯 的 QQ 群有新动作。

在群空间登录以后，进入某个 QQ 群的空间就可以发现多了群邮件、群聊天记录等功能－－不过我很少登录群空间，所以可能说得不对。
群邮件让我想起了 Google Groups，但这里只能够使用 QQ 邮箱。除了这个之外，同时有讨论版和贴图版，再加上群聊，给人的感觉是要查看的地方太多了，势必造成利用率降低。
群聊记录是我一直想要的功能。目前还只支持显示默认的表情，但同步的速度挺快，基本够用了，就看什么时候会有搜索、倒换功能了。

现在整个群空间还处于未竣工状态，听说还有不少地方要改进，虽然对我的吸引力不大，但值得鼓励，毕竟像 Web 页面基本实时地显示群聊记录这样实用的功能，QQ 应该是是第一家实现的。
最后提醒一句，目前只支持 IE，不支持 Firefox 和 Opera。
