三岔口
Table of contents

《三岔口》有李心洁，可惜戏份不够多；悲剧的结果，尤其是说明了李心洁已经在十年前的意外中死亡，而郭富城还满怀希望地开始第二个寻找女友的十年，我是很喜欢的。
其实，除了矫枉过正的郑伊健、枉死的李心洁、爱子心切到发狂的罗嘉良，我觉得其他人物的命运也可以改变一下：
吴彦祖和宁静那种逃脱不了杀手宿命的结局未免太过老套，不如让吴玩火玩得更大一点，直到在他要告诉郭真相之前被当做真凶杀死…
而郭在找不到女友和查不出真凶的折磨中因为偏执和精神失常，对郑做出违反法律的事情，最后自己也入狱服刑…不不不，还是剧中的安排要惨一些。

这次郭富城的表演似乎有了一些进步，果然悲情的主角比较好演绎么？
宁静竟然光头出镜，真伤人心…难道是演完了上一部戏来不及蓄头发？
吴彦祖的角色神经质稍微少一点，光芒立即就减弱了，看来仍需多多锻炼呀。可是一脸邪气的他要尝试那种从心底里散发出让人无法抵挡的正气和纯真的角色，就要看他的潜质和造化了。

回归 Furl
Table of contents

本来 BlogSome 计划只停机20至25分钟，换上内存就完事。但据说到手的内存规格不对，于是系统停机的时间延长。看样子更换的货还没有送到，估计近期还会有一次停机？

提供 Voiz Online 服务的 PhpMore.com 也是一副“503 Service Temporarily Unavailable”的面孔，毕竟不如 365Key 这样的服务有保障。

说到 365Key，由于最近解决了 Furl 的乱码问题，我打算把网络文摘的重心又放回 Furl。
一个是因为 365Key 的非弹出窗口模式收藏之后不会自动回到原始页面；
第二是因为 365Key 的人工自定义、系统规范的两套分类让人感觉很麻烦，因为 Furl 和 Del.icio.us 不会考虑在首页按照系统规范死的分类列出各自最新的收藏，它们只列出最新收藏而不管是什么分类。也许 365Key 是要考虑国内的特殊情况吧，如果照搬那两家，我估计 365Key 的首页便都是“女人叫床 男人叫苦”、“一个‘自慰’女生的秘密”或者“意志的凯旋”、“日货清单”吧。
第三是因为收藏的某些文章可能对 365Key 本身没有好处吧，而使用国外的服务就不用顾虑太多。

Firefox 扩展和实用技巧
Table of contents

公司的机器重装了系统，因为没有作备份，Firefox 的扩展也要重新装一下，因此在这里讲一讲我比较喜欢的扩展和打听到的实用技巧，下次就可以偷懒了：

常用扩展：
1、Adblock
阻挡烦人的广告的好东西，别人称赞得太多，我就不多说了。
介绍|下载安装

2、JustBlogIt!
目前用来在 Voiz Online 上面收集语录。可惜不能同时支持多个站点。
介绍|下载安装

3、Tabbrowser Preferences/Tabbrowser Extensions
方便使用标签的扩展。
安装 Tabbrowser Preferences 以后在“选项”页面中“标签式浏览”的子选项窗口中会多一些选项。而 Tabbrowser Extensions 功能更加强大，以前用过，后来停掉了，因为确实会拖慢 Firefox。不过据说现在已经有了改善了，有兴趣的不妨自己试试：）
介绍|下载安装

4、EditCSS
在侧边栏实时编辑 CSS 和查看网页效果的扩展。Web Developer 也有相同的功能，但是没有这个方便，而且功能过于强大，通常没有必要。
介绍|下载安装

5、Super DragAndGo
实现类似 MyIE 中拖拽的效果。
介绍|下载安装

非常用扩展：
1、Sage
简单的 RSS Feed 阅读器，可以在网页中查找 RSS Feed。但好像没有后台刷新功能，经常占用了当前正在浏览的页面，比较让人头大，不知道现在有没有改进。
介绍|下载安装

2、Show Image
重新载入显示失败的图片。
介绍|下载安装

3、Web Developer
非常强大的网页开发工具，如果你觉得为了它值得牺牲一个工具栏的面积，那就赶紧下载吧。
介绍|下载安装

4、ColorZilla
在页面上选取颜色、调整颜色、缩放页面的工具。
介绍|下载安装

5、FlashGot
配合下载工具使用，增强右键菜单。
介绍|下载安装

实用技巧：
请首先在地址栏中输入“about:config”，打开后再进行以下操作：

1、在“过滤器”中输入 “browser.search.param.Google.1.custom”与 “browser.search.param.Google.1.default”，清除其值“chrome: //browser/content/searchconfig.properties”。
效果：之前在快速搜索栏输入关键字查找后，只会出现 Google 首页，并不根据关键字进行搜索，而现在则会恢复正常。

2、在“过滤器”中输入“browser.tabs.showSingleWindowModePrefs”，将默认值 false，修改为 true。
效果：启用了单窗口浏览模式，“标签式浏览”选项窗口中多出了选项设定。

3、在“过滤器”中输入“network.http.pipelining”和“network.http.proxy.pipelining”，设置为“true”；输入“network.http.pipelining.maxrequests”，设置为8以上。
效果：传说浏览速度会加快。

4、在“过滤器”中输入“browser.xul.error_pages.enabled”，设置为“true”。
效果：载入页面失败的时候不会跳出一个提示框就完事了，而是允许你不断刷新，并且不会弄丢载入失败的页面的 URL。
