1、去掉桌面所有的图标
取而代之的是自己敲命令启动程序。不要以为用键盘就很繁琐，其实比牵着鼠标在一大堆图标、目录当中苦苦寻觅要快得多。

更极端的做法是把“开始”菜单和 Quick Launch 中的快捷方式都删除掉…

需要用到的软件：
TypeAndRun/RunFast
本人是使用前者，有汉化版本，Google 之即可。

学习 Mac OS X，看不顺眼的窗口如果不能关就收缩起来，免得背景一片狼藉。

需要用到的软件：
WinRoll

3、精简任务栏。
把暂时没有活动的程序统统发配到托盘区去！

需要用到的软件：
TrayIt!/PowerMenu

4、设置任务栏为隐藏，能够开辟更多的空间。但我不习惯这样设置。

试用 WordPress.com 的 blog
Table of contents

由于窝子的推荐，收到了 fdcn 发来的 WordPress.com 邀请，在此特别感谢两位！:D

这个 blog 的地址是http://calon.wordpress.com/

刚登入的时候无法更改初始密码，还以为有什么问题呢，结果过了10几分钟，修改密码顺利通过…看来确实是我的 RPWT。

之前就看过新的界面，还是更喜欢朴素一点的灰色界面。Dashboard 内容更加丰富了，虽然我通常只看右上角那一小块。

Write 界面最大的改变就是 WYSIWYG 的编辑器，如果不喜欢，可以在选项中关掉。

Manage 栏目中多了“邀请”一项。WordPress.com 的邀请只有一个，显得比较小气。一旦被邀请者申请 blog 成功，他的 blog 链接中会自动加上邀请者的 blog 地址。一大缺憾是没有 Files 子栏目，不能够编辑文件内容，这样就无法更改模板的设置了…相对应的，Presentation 下的 Theme 子栏目只能够选择不同的主题而已，没有 Theme Editor 子栏目。

Links 栏目中好像还没有提供链接导出功能，虽然自己动手没问题，但毕竟不如系统提供的功能方便。

Options 栏目中没有了 Plugins 子栏目！也无法更改上传文件的路径。看来 WordPress.com 是彻底断绝了使用插件的路子，不知道是不是因为 WordPress MU 是用于多人环境的，所以要安全第一？可不得不说，少了丰富的插件，WordPress 的魅力也就随之下降了很多。

Upload 里面的内容也没有多大变化。可以看到提供的上传空间有25M，传图片恐怕不够，而插件等等又没有用武之地…

虽然现在功能不够丰富，也许今后随着 WordPress MU 的升级，WordPress.com 会提供更多的功能吧，当然，最最要紧的，是不要立马被 GFW 给盯上了…

最后说一句最重要的，手头有一个邀请，不知道朋友中间哪一个真正需要，需要的请留言或发邮件到 calon.xu at gmail.com。优先宿遗的成员，不过我估计都用 MSN Spaces 或 Blogcn 去了，对 WordPress 感兴趣的不多。

Racing Club de Paris #12

逃离降级区看似只有咫尺之遥，却并非那么容易。主场0:0战平排名第9的 Boulogne，不过是帮助对方攀升一位而已，本队依然在降级区的边缘－－而且是靠里面的边缘－－苦苦挣扎。
仅仅出战了3场的 Ravneng 已经进入了最佳阵容，还在考虑是否续签合同的 Izquierdo 也习惯性入选，但本场比赛表现活跃的 Riksfjord 因为脚踝扭伤要躺一个月，我真要庆幸没有把 Bouabdellah 立即甩手卖出去…

这场比赛除了中后卫 Lalisse 的表现有些萎蘼之外，M C 的 Eyjólfsson、Malfleury 两员小将也完全无法承担起中路保护的责任，虽然他们顶着巨大的压力没有发挥失常，特别是 Tackling 不高的 Eyjólfsson 有5次断球，已经尽了最大的努力，但防守属性的薄弱决定了他们在4-4-2两翼齐飞的战术体系中距离双后腰式的 M C 有不小的差距。其实在执教 Reading 时，我最喜欢的阵形还是强调两个 D/DM RL 积极助攻的 3-5-2 与 4-3-3，但目前受人员组成与实力所限，估计要长时间地围绕 4-4-2 及其变体大做文章，所以在有数位边路进攻好手的情况下，M C 位置的综合实力尤其是防守功力就非常关键了，稳固了中场中路，攻，则可与边路互相呼应，支援锋线攻城拔寨，守，则可保护后卫面前的危险地带，缓冲对方的凌厉攻势。

引进了新人之后，进攻效率的痼疾仍然没有解决。目前球队以6个进球名列20支球队的并列倒数第2，总不至于要我在找到合格的正印前锋前把 Bouabdellah 推到锋线上吧？唔，可以考虑考虑，可是那样的话，我的新人就没有什么希望出头了…

就在我满脑子思考后腰和前锋的“关键”问题时，一个“要命”的问题接踵而至…

9月16日，Ronald Thomas，我一直绝对信赖的主力门将，忽然提出要求被挂牌出售！他目前的身价可是只有\$60K 呀！他才刚刚从 Guigamp 转会过来呀！我真的要疯了…拒绝了他的转会要求，立刻和他商谈新合同。Thomas 趁势来了个狮子大张口－－周薪\$2.6K，签字费\$3K，救险奖金\$120，最小转会费条款\$240K！我只好和 Thomas 坐下来慢慢地谈…

最后是唯一的好消息，联赛中表现最好的 D C Izquierdo 终于下定决心和我们续约，合同截止日期是2005年6月12日，届时 Izquierdo 已经有34岁，也就是说如果他期间不转会的话，很可能就要在 RCP 退役了。

可是连这唯一的快乐也是短暂的，9月21日，就在右后卫 D’Amore 从脚伤中恢复过来重新投入训练的当天，左后卫 Soulas 又腹股沟拉伤，将缺席一个月之久…Soulas 刚趴在担架上抬到医务室没多久，Vasseur 又在一声惨叫中撞伤胫骨休战一周…难道我的队伍始终都要保持最少3名伤员么？

乱翻书
Table of contents

上午下午连续开会，中午趁着大家去聚餐，找了个借口跑到书店翻翻书。可惜既没有充足时间深入阅读，也不能买下好几本带到下午的会场上，所以只能够匆匆浏览。

《“好消息”里的“更新”－－现代基督教福音派思想研究》，董江阳著，维真基督教文化丛书中的一本，定价24元。因为只专注于在北美和我国最泛滥的福音派，所以比较值得一看。

《批判哲学的批判－－康德评述》，李泽厚著，28元。瞟了几眼感觉还不错，虽说是名家写的评论名家写的名著的名著，入手之前还是得多留个心眼的好。

《理智设计论－－科学与神学之桥》，威廉·邓勃斯基，25元。书名与作者名字翻译上的不妥当就让人很不爽。看了其中一段介绍如何判断是否具备智慧的文字，给我的感觉是很不严谨，其标准甚至还不如用来判别人工智能的图灵测试，当然，也可能是我还没有看到图灵测试相关的章节，但我估计作者还不会笨到引入图灵测试，因为当下既没有交互的条件，也看不出有通过测试的希望。
与它同属清华哲学翻译系列－－北美宗教文化专集的还有《福音派与基督教的未来》、《认识美国基要派与福音派》两本能够激起我的兴趣，但当时懒得多看看了。

《控制论与科学方法论》，金观涛、华国凡著，16元。遗憾的是没有找到有出同系列的信息论方面的书。

《科学、迷信与社会》，迈克尔·波兰尼，17元。最好和他更为成熟的《个人知识》一同阅读，不过还是多看看科学书籍之后再来思考他所提出的问题吧，因为从“科学的绝对客观性标准”、“绝对本质和实在”等词句来看，我认为他对还原主义甚至科学本身的理解有不当之处，用并不适用于科学的标准来要求它批判它，而且在当下打着中国传统文化和后现代主义大旗的“文化圈”，波兰尼的理论属于最容易被利用和误读的那一类，吸收和批判的时候一定要慎之又慎。

《科学的规范》，卡尔·皮尔逊，19元。要注意作者对斯宾塞/社会达尔文主义的错误态度，另外这类年代比较久远的科学书籍的一大缺憾是没有赶上物理学大发展的时期，不过大部分内容直到今天都是很有价值的。
