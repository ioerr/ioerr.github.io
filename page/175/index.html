谈谈评价歌手的标准
Table of contents

无语，本来写了很多，觉得大多是废话，又删减了。

谈一谈我评价歌手的标准。
主要分为三个方面：天赋、技巧和悟性。

天赋，一般指的是硬件条件，比如说音色、音域、力度等等。天赋条件好的歌手起点就比别人高出很多，一开始会比较讨巧，以后要达到和别人同样的高度也可以少奋斗 XX 年。但依靠后天训练的技巧与自身的悟性，可以弥补一部分天赋的差距，甚至超越那些天赋更高的歌手。当然，少数上天的宠儿的某些方面是其他人后天怎么努力也赶不上的，拿音色来说，有的人一张口一出声就让人如醍醐灌顶，如醉如痴；相反，有的人的音色单薄到给人一种有心无力的感觉，或者尖锐、毛糙到刺耳的地步，也是无可救药的…

天赋较差的例子：音色和音域都不够好的张艾嘉，破铜锣嗓的 Rod Stewart、陈昇。
天赋过人的例子：以啸叫为特征的 Vitas、Mariah Carey、Josh Groban
存在争议的例子：阿杜

技巧，后天修炼来的独门法术，达到普遍水准之上即可以脱颖而出，弥补天赋的不足。如果修炼成精，便能自成一派，进而凌驾于一般的靠天赋吃饭的歌手之上。

技巧独特的例子：邓丽君、王菲、李谷一
几乎没有技巧的例子：罗大佑、张艾嘉
存在争议的例子：周杰伦

悟性，这是一个很模糊的词语，不太好讲明白，大概相当于武侠小说中同名术语，只可意会，不可言传？
乐感、节奏感等等很多素质都可以看作是悟性的反映。这方面突出的歌手在处理自己并不熟悉的歌曲时，往往能够凭自己的理解对节奏、做出最合理的调整，在进行技巧训练时也能够做到事半功倍。同时，他们更擅长于认识自己和理解歌曲，往往懂得选择适合自己特色的歌曲，又能明白怎样做才能最有效地表现歌曲，因此，这个方面超过常人的歌手即使天赋与技巧稍逊一筹，其表现力也丝毫不落下风。另外，他们的演唱水平会随着训练与表演而突飞猛进，可能会超过单纯有天赋或技巧优势的歌者。

有悟性的例子：庾澄庆、伊能静、天赋和技巧都不够仅能够说音色有特色的莫文蔚
悟性不够的例子：天赋和技巧都不够悟性更不够的徐怀钰、陈慧琳

但其实最重要的是一个歌手的风格与他/她选择的歌曲的内涵、境界是否浑然天成，也就是说，他/她的表现能够最大限度的体现他/她作为一个歌手的艺术特质，并且这种特质能够达到打动人心的境界。
所以有的时候，你会觉得某人唱的歌实在是难听，嗓子破，唱功也是平平，可过了很长一段时间之后，反倒觉得这才是表现最佳的版本。
许多创作型歌手虽然天赋条件和技巧都只算是差强人意，但演绎自己的歌曲却是别人难以超越的。最好的例子就是罗大佑的《野百合也有春天》、《你的样子》被林志炫翻唱，林志炫无论天赋还是技巧都远胜罗大佑，但是翻唱这两首歌的效果只能够哄哄冲着声音去的人－－而他翻唱张学友的《离人》又超过了原唱。

正面的例子：罗大佑、陈珊妮、陈昇、Lennon、Bono
反面的例子：目前的张惠妹、王力宏

在我看来，除了上面这种例外情况，评价歌手时悟性最为重要，天赋次之，技巧再次之，当然这个排名并不绝对，只能说是一种大体上的倾向。

以上仅就较为通俗的歌手随便说说，所举的例子可能还有不合适的地方，比如说，好几个人选我都不能够确定是否属于有悟性，因为这东西太难界定了。

Google Talk 实用技巧
Table of contents

/nomutex 参数，可同时启动多个进程，方便测试
/prefetch=1 传说能够加快载入速度
/superprefetch 传说能够大大加快载入速度，这两个技巧要慎用，反正现在启动也挺快的

使用 ResHacker、 ResEdit 或者 eXeScope 能够自己修改 googletalk.exe，例如增加按钮、托盘菜单选项和修改界面文字

在文字前后增加*符号，可以实现加粗效果，增加_符号，可以实现斜体效果
如 *bold*：bold
或 *italic*：italic

Google Talk
Table of contents

下载

好话就不多说了，界面简洁，功能简单实用，正是我要的 IM。如果有群功能，就可以抛弃 QQ 了。

说说到目前为止发现的不足：
1、没有群功能
2、没有分组功能
3、表情、截图、文件传送这些的都可以不要，但字体应该允许修改吧，还好目前的字体我很满意
4、体积虽小，占用的内存却不算少，一般都是10几 M

希望有什么改进：
1、以上数点
2、新增加功能的话，最好是做成功能模块，允许用户自己选择加载什么模块。
3、直接保存聊天记录到邮箱，另外，虽然它说有保存最近的聊天记录，但找不到在什么地方查看
4、要能够同时多开几个就好了…
5、快出中文版吧: )
6、最好能够选择进一步精简语音通话功能，因为我几乎不用语音通话
7、能够选择始终保持最上层吗？太喜欢这个淡雅的界面了，Google 的 UI 设计万岁！

闷棍
Table of contents

Zizou 回归了，法国队3:0大胜象牙海岸，Domenech 先生虽然面上高兴，心里还是对“青春泉作战”的半途而废颇为遗憾的；数天之后，另一位国家队教练席上的失意者 Santini 先生的心情恐怕就不只是“遗憾”二字可以形容了。

从功成名就的 Lyon，到败走麦城的法兰西，到两面不是人的 Tottenham Hotspur，Santini 走的是一条不断向 X 轴靠近且斜率小于0的曲线。好在 Auxerre 的传奇主教练 Guy Roux 这个赛季宣布退休，给了他第二次东山再起的机会。

球队失去执教了44年的教练，受到的影响也许不亚于 Sir Alex 离开 Man Utd，Silvio Berlusconi 辞去Milan 的主席一职吧。不过 Santini 的接手工作似乎还不太坏，虽然杯赛1:4惨败给 Lyon，但联赛开始后2胜1平，包括连克 AS Monaco 与 Bourdeaux，紧随 PSG 占据第2名，而且后防线不失一球，相信 Guy Roux 看到也会放心吧：“这小子，在法甲还是混得开的”。

然而人有脑后闷棍，哦不，旦夕祸福，天有不测风云，8月20日晚上 RC Lens Bollaert 球场的阴冷小雨让 Santini 深刻体味到了这句话的苦涩。Lens 人几乎每一次射门都能够洞穿 Auxerre 的大门，0:7的比分创下了俱乐部参加甲级联赛以来的历史，估计最后阶段 Santini 只能够像2002年夏天那样无助地坐在教练席上瞪着一双失神的眼睛发呆吧，丧失对球队的控制对于他来说已经不是一种陌生的经历了。而这再一次证明自己的机会忽然成了写下又一笔职业生涯耻辱的石碑，被命运狠狠戏弄过一次的他还有机会还有信心再待下去吗？

Younes Kaboul、Bacary Sagna 受伤病困扰无法出场也许是 Santini 唯一的借口，可0:7的闷棍实在对不住俱乐部的上上下下，尤其是他们才刚刚告别可敬的 Guy Roux，两相对比之下，Santini 的无能又被放大了数倍。更要命的是，下一场比赛正是老东家新霸主 Lyon。是的，Lyon 似乎开始慢慢染上了 Gerard Houllier 综合症，不过根据墨菲定理，如果事情有可能变糟，它就真的会变糟，所以 Lyon 大开杀戒再来个4:1也不是没有可能。再往后，是数支中游球队，实力一般却最为难啃，挺不过去的话，Santini 先生的 Auxerre 百日游也就到此为止了吧。
