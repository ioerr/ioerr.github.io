招安
Table of contents

我们现在身处中国，就要按中国的传统及习惯讨论问题。作为文学，不论古代，还是现代，必须为主流所承认，必须为“在朝”者肯定，才能登堂入室，才能被称为文学，其创作者也才能被谓之“名副其实”的作家。并不是芸芸众生、在野之乌合，齐声呐喊几声，就可称王称霸，继承大统。真要是这样，宋江也就不用搭进性命，力争招安了。
多么彪悍的文字，不愧是世交，真乃知己呀，简直是现代版俞伯牙锺子期！

甭管你的文字是不是受大众的欢迎吧，取悦了他们，那叫低俗，取悦不了他们，那叫没水准。

只有顺着文坛主流前辈给指明的道路、得到高高在上的“在朝”者的肯定，你才有资格进入文坛，你写的东西才叫做文学，你就能够名正言顺地强奸文学及文学的妈啦～

原来这种光荣的事业，古时候是叫做“力争招安”呀。宋朝那个叫做宋江的小吏最擅长干这个，具体一点说，他的专长是“给朝廷命官松绑，纳头便拜，口称死罪”、打着忠义的旗号往兄弟的两肋插刀，以及锲而不舍地提供一步到位的打包招安服务。

当然，除了“力争招安”之外，他还会把其他那些不鸟主流、没心思受招安的“在野之乌合”都做掉。不过王先生不提这一点，可能是觉得韩寒受招安方面的天赋比较低，现在搬上议事日程还太早了。

另外，王先生既感慨于当下社会“泛文化”现象严重，不妨多宣传宣传这种有光明前途的“招安文化”，多多教育一下在野之乌合，让他们明白识时务者为主流，赶紧买脑白 X 给文坛前辈赔不是，下次再遇到有不识抬举的小辈儿，要不遗余力地帮忙猪圈，哦不，是助拳，直到帮助他们认识到招安文化的先进性和招安事业的伟大为止，届时必可使本社会的风气噌地焕然一新，道德水平刺儿溜翻一番，主流文坛也必然和天下之大谐了。

MultiGo
Table of contents

以前试过几个围棋打谱软件，有的操作不方便，有的格式互不兼容，有的还要收费，当然，我对围棋算不上狂热，也许还有很好的打谱软件没有找到。不过 MultiGo 这个对我来说已经完全够用了。

Multio 是免费的软件，很可惜既不是开源的，也没有移植到其他平台的打算。
它支持的棋谱格式包括：
BDX (清风)
NGF (新浪)
GOS (联众)
GIB (棋圣道场 Tom.com)
UGF (IGS Panda Egg)
GO (中国游戏中心)
功能强大，各项设置也挺贴心，以前我还没注意过可以设置是否按应氏围棋基金会的坐标法标示坐标呢。
配合开源软件 GNU Go 还可以实现人机对弈。据说棋力在18级，不过像我这样水平只够看棋的，打谱看看高手们的对弈就够了…

五子棋中的八卦阵…呃，放到19路棋盘上不会有人砍我吧…

入手《宇宙尽头的餐馆》
Table of contents

终于等到天气不错的周末，抓紧时间去了一趟书店。

买了 Richard Dawkins 的《盲眼钟表匠/盲人钟表匠/The Blind Watchmaker》，不算是新书了。他的《解析彩虹——科学、虚妄和玄妙的诱惑》怎么也找不到了，是不是借出去了？

在图书城的科幻书柜找到了几本经典科幻作品，但是统统不全！
《基地》三部曲缺少第二部；
《沙丘》三部曲缺少第一部；
《光晕》三部曲缺少第一部；
《银河系漫游指南》也缺第一部…
还好《科幻世界》在正刊和译文版上有刊出过《银河系漫游指南》、《光晕》的第一部，所以我就买下了《宇宙尽头的餐馆》、《光晕：洪魔》和《光晕：初次反击》。

《银河系漫游指南》系列不用多说了，虽然不属于硬度十足的科幻，但绝对是可读性最强的经典科幻小说之一，冷笑话层出不穷的同时也回味无穷。今天吃晚饭的时候险些被它杀死…
另外提一点，Douglas Adams 曾经应邀创作了电视连续剧《Monty Python’s Flying Circus》第45集的剧本，而 Python 语言的命名正是取自 Monty Python，Spam 一词也来源于 Monty Python 的短剧。

《光晕》嘛，虽然我没有XBOX，也没有玩过 Halo，但其第一部《光晕：致远星的沦陷》让我沉浸其中饥渴地阅读了两天。那种实力悬殊，以至于让人深深地绝望的感觉刻画得非常好，即使主角的设定属于我不喜欢的超人精英。
《光晕》第一、三部的作者、翻译与第二部的都不相同，但愿水准也保持一致。

不是人
Table of contents

说起“不是人”，自然就联想到陈某的三国志主题漫画。不过这一篇文章和漫画无关。

最近三国志 XI 上市，伍岭正玩着呢。不过怀旧的我还在鏖战三国志5+罗蔚旻的补丁98，熟悉这款作品的人应该清楚，吕温候的表现确实对得起“不是人”这三个字。

吕布

武力100，方天画戟加武力10，头像附加隐藏武力16，勇猛7，冷静0，单挑必杀技无法防御，加上坐骑为赤兔，要跑路很困难，基本上不要妄想单挑。

勇名满，偃月阵（此阵勇名低时很一般，勇名高才有威力），出手往往就是一把大叉－－会心一击，重创…

武将技能如下：
回射：如果你向他射箭，他立马回射你。你伤他几十几百人而已，他射回来一个重创，你就惨了。
远箭：弓箭射程增加一格。
骑射：移动命令之后仍然可以射箭。
乱射：向周围的部队无责任乱射箭。配合远箭，人人有份遭殃。
奋发：同时攻击四面的敌方部队。又是一片重创…
无双：消除敌方一齐攻击的附加伤害，团团围住也拿他没辙。

再加上占城加防御，每回合回复伤兵，你攻他如隔靴搔痒…

光是描述数值和技能没有说服力，还是以这一次开局的游戏中实际战例说话：

197年9月，袁绍65000人进攻吕布的陈留，宋宪、陈宫被击溃，张辽单挑败给颜良。颜良、审配、辛毗、刘辟、张郃把吕布团团围住，于是吕布出手了…
一个回合的会心一击就把颜良的部队消灭了，然后两次奋发，剩下的人兵力所剩无几，统统退兵了…

199年12月，袁绍发兵9万进攻吕布的陈留，吕布自濮阳驰援，加上陈宫、张辽、高顺的部队总共55000人左右。

陈宫跑去劫袁绍的粮仓，结果遇到回护的审配、文丑和增援的2万袁绍军，分分钟就被搞定。

高顺等人也没有撑多久，又变成袁绍诸将围攻吕布一人的局面。
吕布乱射射翻淳于琼、审配、辛评，奋发同时击溃颜良、文丑，张郃、沮授根本还没交手就退兵了…

1个月后，200年1月，大耳贼打算浑水摸鱼，集结了6万人进攻陈留，一上来吕布就在单挑中把关二爷砍成重伤…剩下的张辽、高顺就搞定了。

200年8月，袁绍又来了，先头部队就有7万人。
高顺利用城池堪堪耗尽颜良的兵力，旋即被袁煕、沮授的夹击战退。
张辽单挑险胜文丑，但是还是挡不住后面的大部队。
陈宫又想去劫粮，正好撞上袁绍的5万援军，袁谭放一把火，袁绍来一个伏兵，又是分分钟被搞定…

于是吕布又出手了，第一回合就把张郃整支部队灭了，第二回合袁谭没了，第四回合审配束手就擒，第六回合沮授崩溃了…袁绍见势头不对，直接跑路…
