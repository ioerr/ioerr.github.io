新增 VeryPlainTxt 主题
Table of contents

一直想给臃肿的首页减减肥，虽然侧栏增加了收缩效果，但对减小体积没有助益，只是版面稍微整洁了一点，所以还需要动大手术。

正好看到 Plaintxt.org 有一系列的 Plaintxt 主题下载。因为我个人很反感三栏布局，没有采用三栏的 plaintxtBlog，而是选择了传统两栏的 veryplaintxt。

顺便找到适合 WordPress.com 2.0+ 的 Theme Switcher 插件，很快就用上了。

之前修改默认的 Kubrickr 主题时，没考虑太多的语法规范，结果无论是页面还是 CSS 都慢慢变成一团乱麻，看着就头痛，遂懒得回头整理。这一次修改 CSS 样式、页面的时候就规矩多了，不敢说完全符合规范，至少比原来有了很大进步。

这个主题尽量不使用彩色，不使用图片，少用 Javascripts，侧栏大部分的内容都进行删减或转移到了独立页面中。
使用方法是，在侧栏找到 Theme Switcher，点击主题名称。如果切换主题有问题，请删除对应的 cookies。

如果反响不错，我会把这个主题作为默认主题。

我的 WordPress 插件
Table of contents

响应 Blogging Pro China 号召，列出我的 WordPress 插件，顺便庆祝 Weblogs.us 解封！

1、Adhensive
用于设置置顶帖。功能简单，但挺有用的。
由于不兼容2.x 版本，已经删除。

2、Akismet
防杀 Spam 的工具。
由于会影响 Weblogs.us 的稳定，因此被删除。

3、Better Feed
增强 RSS Feed 内容。

4、Extended Live Archives
可以实现 Live Archive 功能。

5、FancyTooltips
用 Javascripts 美化 Tooltip，把鼠标移到链接上就可以看到效果。可以手工添加，但插件形式要方便一点。

6、Random Quotes
如侧栏所示，用来随机显示摘录的言论。

7、SearchNow
即时显示搜索结果，见侧栏最上方。有时候会没有动作，需要多输入一个空格。

8、Subscribe To Comments
留言者可以订阅回复。

9、Twilight AutoSave
暂时保存在线输入的文章内容，防止丢失。
由于有良好的保存习惯，所以基本没用到，已经删除。

10、Ultimate Tag Warrior
强大的 tag 工具。
由于觉得麻烦，所以删除。

11、WordPress Database Backup
用于数据库备份。

12、wp-Define
以 Tooltip 形式给指定的专有名词加上注释。
懒得维护，所以干脆删除。

13、WP Grins
在回复对话框前增加可点击的头像。

14、中文 WordPress 工具箱
显示最新回复、随机文章等等。

15、Dean’s Permalinks Migration
无痛转换 Permalink 结构。

16、No Ping Wait
提高发布时 ping 的速度。

17、Spam Karma 2
由于 Akismet 无法使用，所以用这个来对付 spam。

18、SRG Clean Archives
产生一个简单的文章汇总页面。

19、Theme Switcher
快速切换主题。目前还有一点兼容性问题。

20、WordPress.com Stats
官方提供的简易访问统计工具。

21、Editing Manager
为修改过的文章增加时间戳，同时可以控制是否在Feed中更新。

TOR 简明教程
Table of contents
    • 1、下载：

近日成功地拉拢我的表妹成为本 blog 的读者，因为要访问这个 blog 就需要突破 Great Fire Wall，所以教她学会了用 TOR。顺便放出简明教程。

1、下载：

的主页已经被屏蔽，所以没有办法直接下载到。可以到这里下载 TOR + TorCP + Privoxy 打包安装程序。感谢 Google Pages 提供的服务！

如果今后有重大升级，我可能会更新下载地址。

2、安装：
没什么好说的，一步一步照着提示做就成了。

3、运行 TOR：
实际上是运行图形界面的 TorCP 和具备过滤内容、转换代理功能的 Privoxy。
从“开始”菜单-“所有程序”中找到 TorCP 和 Privoxy 并运行即可，也可以设置批处理命令或用 TypeAndRun 启动。
注意：打包安装版可能会设置 TOR 随电脑启动运行。

4、设置浏览器代理：
如果使用 IE，在“工具”-“Internet 选项”-“连接”-“局域网（LAN）设置”-“代理服务器”中，钩选“为 LAN 使用代理服务器”。
然后在“地址”一栏中填写表示本地 IP 的127.0.0.1，端口号设置为8118。

如果使用 MyIE/Maxthon，应该在工具栏上有切换代理的工具，老老实实地填写 IP、端口，启用即可。

如果是使用 Firefox，推荐使用 xyzproxy 插件。安装后在工具栏上 xyzproxy 图标的下拉菜单中选择“工具”-“编辑代理”，输入127.0.0.1:8118#Local。然后点击图标，使它由灰色变成彩色，便启用代理了。

5、其他说明：
理论上到上一步就能突破封锁了。可以用浏览器打开这个 blog，或者 WordPress Documents、维基百科、新语丝、Google 的网页快照、BBC 中文网、中国网络审查条目等来测试是否成功。

如果通过 TOR 使用 WordPress 时出现和“Enable Sending Referrers”相关的错误信息，你就需要编辑 Privoxy 的 User Actions 文件，在{ fragile }起始的一段中增加问题站点的地址，如 .weblogs.us
