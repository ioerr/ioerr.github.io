TOR 简明教程
Table of contents
    • 1、下载：

近日成功地拉拢我的表妹成为本 blog 的读者，因为要访问这个 blog 就需要突破 Great Fire Wall，所以教她学会了用 TOR。顺便放出简明教程。

1、下载：

的主页已经被屏蔽，所以没有办法直接下载到。可以到这里下载 TOR + TorCP + Privoxy 打包安装程序。感谢 Google Pages 提供的服务！

如果今后有重大升级，我可能会更新下载地址。

2、安装：
没什么好说的，一步一步照着提示做就成了。

3、运行 TOR：
实际上是运行图形界面的 TorCP 和具备过滤内容、转换代理功能的 Privoxy。
从“开始”菜单-“所有程序”中找到 TorCP 和 Privoxy 并运行即可，也可以设置批处理命令或用 TypeAndRun 启动。
注意：打包安装版可能会设置 TOR 随电脑启动运行。

4、设置浏览器代理：
如果使用 IE，在“工具”-“Internet 选项”-“连接”-“局域网（LAN）设置”-“代理服务器”中，钩选“为 LAN 使用代理服务器”。
然后在“地址”一栏中填写表示本地 IP 的127.0.0.1，端口号设置为8118。

如果使用 MyIE/Maxthon，应该在工具栏上有切换代理的工具，老老实实地填写 IP、端口，启用即可。

如果是使用 Firefox，推荐使用 xyzproxy 插件。安装后在工具栏上 xyzproxy 图标的下拉菜单中选择“工具”-“编辑代理”，输入127.0.0.1:8118#Local。然后点击图标，使它由灰色变成彩色，便启用代理了。

5、其他说明：
理论上到上一步就能突破封锁了。可以用浏览器打开这个 blog，或者 WordPress Documents、维基百科、新语丝、Google 的网页快照、BBC 中文网、中国网络审查条目等来测试是否成功。

如果通过 TOR 使用 WordPress 时出现和“Enable Sending Referrers”相关的错误信息，你就需要编辑 Privoxy 的 User Actions 文件，在{ fragile }起始的一段中增加问题站点的地址，如 .weblogs.us

推荐 Firefox 扩展与主题
Table of contents

扩展

加粗的扩展和主题是目前正在使用的。

2016年5月21日更新：

Beyond Australis

在 Australis 主题基础上进一步优化。例如：
进一步精简顶部工具栏；
隐藏书签工具栏，鼠标移上去才显示；
智能地址栏集成搜索功能，并自定义显示提示信息；
有很新颖的天窗功能，精简书签、网站身份信息界面。

Download Status Bar

之前的 Download Status Bar 不支持新界面，而新版本不但兼容新的 Firefox，而且功能大大增强。
最主要的作用还是在底部状态栏直接显示和操作下载项目。

DownThemAll!

多线程下载、过滤下载清单等定制功能仍然无与伦比。

FoxyProxy

可以自定义代理生效规则，支持地址白名单或黑名单列表，结合 Firefox 的独立代理特性，堪称核心优势。

Instaright!

用于收藏页面内容到 Instapaper。
虽然 Instapaper 被 Pocket 风头盖过，但 Instaright! 最大的好处是可以保存选中的部分，而不是完全交给服务端解析页面的结果，对付一些 Pocket 不能正常处理的页面有奇效。

LastPass

非常方便的密码统一管理工具，适合懒人。

Lazarus

用于保存和恢复已经输入但没有提交的文本内容，多次挽救了不慎关闭浏览器或程序崩溃、死机时未写完的文字。

OneTab

概念像标签页群组书签和稍后阅读类服务，但只是保存页面链接清单到一个本地的标签页。
好处是大大精简标签页，而且批量操作更加快捷方便，还能排除指定网页、锁定和明明标签页群组、导入导出、共享清单等等。

Pocket

前身是 Read It Later 的稍后阅读类服务，跨平台和同步效率体验不错，后来居上超越了 Instapaper。

Tab Tree

自从 Side Tabs 之后就习惯将标签页放置在左侧栏，但 Side Tabs 已经停止更新并且有兼容问题，于是找到了功能更强的 Tab Tree，在侧栏显示标签页之外，还能显示标签页父子关系。

Tagpacker

之前有介绍过，取代 Delicious 的社交书签服务。

VimFx

如果嫌 Vimperator 等扩展太重，Vimkey 等软件功能不够，可以用这个支持自定义快捷键的扩展。

Wapplyzer、Firebug、Hackbar 等适用性不广的扩展就不做介绍了。

2006年5月6日更新：

自从 Flickr 被部分和谐之后，Access Flickr!就派上了大用场。

虽然 Adblock 有导致内存泄漏的嫌疑，但功能如此强大，还是继续用好了。
更新：目前基本不需要 Adblock 了，所以没有安装。

经常复制文字内容，既懒得老是按下 Ctrl+C，又不想像 UNIX 命令行下那样选中即复制，那么选中之后会暂时弹出一个小菜单供你执行复制操作的 Auto Context 就是选择之一。

Autocomplete Manager 能够增强 Firefox 的地址栏的匹配功能，访问历史和书签的标题、 URL 都可以模糊匹配，不过过渡到 Firefox 3之后，这个扩展的魅力值应该会下降了。

虽然 Backword 的即时翻译功能不是最强的，但是保存上下文用于复习的能力是其他扩展不具备的。

由于在 Gmail 中标记已读经常误操作，所以用了集成很多脚本的 Better Gmail，还有右键预览、附件图标、集成 Google Reader 等多种功能。

询问 Windows 下的 IRC 客户端，回答都是 mIRC，可是 mIRC

是共享软件啊…所以还是用 ChatZilla!

更新：目前非要用到 IRC 不可才把 ChatZilla! 激活。

经常复制链接就需要 CoLT，自动配置格式，且可以自定义。

Diigo 的专用扩展 Diigo Toolbar 比 Diigolet 书签按钮稍微好用一点。

Del.icio.us 的官方扩展 解决了页面跳转和搜索书签的问题，让 Del.icio.us 变得更好用。

Download Status Bar 解决了我打开下载清单窗口速度慢的烦恼，删除文件也更方便。

DownThemAll!

号称 Firefox 专用批量下载软件，大部分时候都能够取代 Flashget，有时也会出现无法正常下载的问题，或者因为不方便更名、对中文支持不好而给用户造成困扰。

有时候还是需要靠拖动来节省时间的，比如快速搜索、在后台标签打开 URL、图片等等，所以又用上了 Drag de Go，花时间设置一下就挺好用了。

FishEyeTabs 的效果是，鼠标移动到标签上，标签就会自动展开，适合标签很多很拥挤的情况。不过初上手可能会有点不习惯。

用 FlashBlock 可以阻止下载未经允许的 Flash 动画，页面一下子变得非常清静，而且还节约了系统资源。

一直很奇怪为什么 Flashgot 总是名列 Firefox 插件评分排行榜首位，可能因为现在我基本都不用常规手段下载文件了。前一阵子转用 Wget 代替 Flashget 和 Net Transporter，所以才安装了 Flashgot。
更新：大部分时候使用 DownThemAll，偶尔用一用 Flashget/Wget，也就不必常备 Flashgot 了。

FootieFox，适合足球爱好者的扩展，自动读取当日欧洲足球比赛的比分信息，几乎是实时显示。前一段时间升级之后，改进了显示比分的方式，不用点开一场场比赛查看比分了。
不足之处也很明显，例如只能够选择一个喜爱的队伍或赛事，不能够自定义屏蔽某些小联赛的信息，有的时候会漏比赛或比分等等。
更新：右下角消息太多了，所以让它暂时下岗。

有时候 TOR 的速度比较慢，如果只要使用一般的网页代理，那么 [gladder** 就够用了。

通过无穷尽的脚本使 Firefox 的功能有第二次扩展的机会，这就是 Greasemonkey。安装之后请去 UserScripts 寻找中意的脚本。

Hide Menubar 用于隐藏菜单栏，是扩大可视面积，精简 Firefox 界面的好工具。
有了 Personal Menu，这个就可以消失了。

遇到严重不符合标准而无法用 Firefox 正常打开的页面时，权宜之计是使用 IE Tab 和 IE View Lite。不过这样一来就更要注意安全问题了。
更新：常用的网站基本都能够用 Firefox 正常浏览，所以不用 IE Tab 了。

遇到过大或者过小的图片就用 Image Zoom 来对付，有丰富的自定义选项。

我懒得手工填写表单，又不放心全部交给浏览器自动完成，所以 InFormEnter 挺适合我。可惜用来存储密码好像太不保险。

调用外部文本编辑器编辑页面文本框内容的 It’s All Text! 可以让你走到哪里都使用自己最爱的文本编辑器，同时还具有临时备份功能，防止文本丢失。

要完整地和长久地显示 Tooltip 提示信息，Long Titles 才是王道。

如果你想依次看编号为1.jpg 至99.jpg 的图片，既不想下载又懒得手工修改地址，怎么办？
Maxthon 利用插件可以实现，Firefox 利用 NextPlease! 扩展同样可以实现，而且功能强大得多…
更新：使用机会越来越少，移除了。

Opanda IExif for Firefox 是查看图片 Exif 信息的好帮手。

精简合并菜单可以使用 Personal Menu，留下常用的菜单项，其他的统统去死！

Personas 是轻量级的主题方案，好处是不用重启就可以更换，缺点是只能更换工具栏、状态栏的背景配色和图片，而且目前好看的主题不多，仅有 Viva Firefox 这一款还不错。

截取页面图片目前应该是 Pearl Crescent Page Saver 最棒。Winson 曾经推荐过 Screen Grab!，虽然可以截取框架图像，但需要 Java VM…
更新：很少用到，暂时禁用。

Save Image in Folder，如果经常保存大量图片到不同的目录，感觉来回切换目录挺烦人，那么这个扩展就很有用，否则的话不必试了。
更新：很少用到，移除了。

为了方便的重新启动 Firefox，于是用上了 Save Session，重启之后能够恢复上一次打开的页面。

ScrapBook，保存网页到本地的扩展。虽然有了Diigo、Del.icio.us 等在线的收藏服务，但本地收藏还是需要的，而且还能够用于在线编辑时做临时存档用。这个扩展在保存时可同时获取指定类型的链接目标，如图像、声音、压缩文件等，可以自定义的选项比较丰富。

裸奔版 Firefox 中如果图片载入失败只好刷新整个页面，很不方便，好在还有 Show Picture。

Status Buttons 用于将工具栏的按钮拖到状态栏上，可以为地址栏腾出更多的空间。

因为实在忍受不了一些人的页面内嵌自动播放，所以不得已安装了 Stop Autoplay。

Stylish 是一个和 Greasemonkey 相近的插件，主要用来改变网页和浏览器的界面。在 UserStyles 可以下载脚本，如 Icon identifiers for links redux。
更新：应用不够 Greasemonkey 广泛，暂时禁用。

升级 Firefox 到1.5之后 Super DragAndGo 的页面一直说版本不对，不能更新，好在 Mozilla Update 中国有适合 1.5.x 的版本可以用。
更新：很少用到，移除了。

有了 Tab Mix Plus，就可以彻底告别 Tabbrowser Preferences/Tabbrowser Extensions。拖拽 URL 到标签栏，可以在新标签页中打开，代替了部分 Super DragAndGo 的功能。
如果不需要那么强大的功能的话，Tab Mix Lite CE 也可以了。

由于常用 Twitter，所以安装 TinyUrl Creator 进行配合，更加方便地缩短过长的 URL 。

使用 Torbutton 配合 Tor 突破网络封锁。

使用 VideoDownloader 下载 Youtube 等视频站点上的视频，更主要的原因是使用多线程下载的话速度会快一点。

编写网页者必备利器 [Web

Developer](https://addons.mozilla.org/firefox/60/)。安装之后功能太多，谨防弄花眼

更新：编辑网页时才用到，暂时禁用。

再次提到 xyzproxy，比 SwitchProxy Tool 方便易用且 bug 少。
更新：试用了 Torbutton 之后就禁用了这个。

主题

2016年5月21日更新：

因为尽量精简了 Firefox 界面，因此基本没有使用主题了。

2006年5月6日更新：

之前介绍了 SomeOrbitYellow、Saferfox Xpanded 和 Plastikfox Crystal SVG 几个主题，后来改为下面几个：

Geek 主题很简洁，但已经不支持新版本 Firefox。已经卸载。

Brushed，借鉴 Apple Mac OS X Brushed Metal 的主题，若没有大片空白，视觉效果还是不错的。
更新：细节不够好，已经卸载。

顾名思义，GNOME Firefox Theme 是仿 Gnome 风格的主题，蓝色的风格更好看一点。

hmmXP 主题基于 HmmXP Visual Style 主题，非常简单的风格，但是很耐看。

iFox，又一款借鉴 Apple Mac OS X Aqua 风格的主题。
更新：“转到”按钮太小，已经卸载。

Modern Pinball，它的按钮让我联想到了复印机等办公设备上可爱的设计…

Mostly Crystal，图标源于 Crystal SVG。
本来是打算在 Thunderbird 也用同风格主题的，虽然比华丽有余的 Tiger Mail 要好，但却比不上很耐看的 CrossOver，反正用上后者之后就不打算更换了。

顾名思义，Mozilla.org 这个主题就是使用的 Mozilla.org 站点的风格。
更新：细节做得不够好，已经禁用。

Netscape Windows 3.1 适合怀旧的人使用，Windows 3.1 + Netscape，别有一种原始粗犷的美…

仿 Ubuntu Human 风格的一款主体 Ubuntu Human Theme，如果用过 Ubuntu 应该会感到比较亲切吧。

Whitehart 既有 Firefox 也有 Thunderbird 版本，风格简洁，自定义选项多，目前是我的 Firefox 首选主题。
