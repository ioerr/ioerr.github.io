KooMail 或 ThunderBird
邮件讨论

今天把公司电脑上的 ThunderBird 用覆盖旧版本方式升级到了 1.5，结果安装完毕之后竟然无法启动，重启之后依然没有解决问题。按某人的话来描述就是：

We uninstalled everything and reinstalled fresh. No go. We rebooted. No go. We uninstalled, rebooted and reinstalled. No go.
When one double clicks the icon, nothing happens. Task manager shows nothing. No windows. No event logs. No pop-ups. No sounds. No chicken nuggets. Nada.
用户体验可谓极差，安装之前也不打一声招呼。
于是临时找来 KooMail 作替代品。

KooMail 占用的资源不算太多，响应比较快，据说 DreamMail 的响应速度就很成问题，程序开得多的话，我的 ThunderBird 也经常反应迟钝。

远程管理的功能不错，可以在服务器上设置废件箱、草稿邮件箱、已发邮件箱。不知道其他客户端能不能够做到。

其他的功能嘛，和 FoxMail 和 DreamMail 都大同小异。自动分割大邮件、分析 IP 地址、建立虚拟子邮件夹的功能不错，但并不新鲜，而且无法建立多级虚拟子邮件夹，听说同时应用多个搜索条件也存在问题。

RSS 功能比较鸡肋，手工删除安装目录下的 RSS 目录或者在查看中禁掉即可。

过滤垃圾邮件的功能比 ThunderBird 差远了，和 FoxMail 比也不是一个档次。

看到作者挺乐于接受用户的建议的，于是去论坛留了一大段话，希望能够改进。

当然，后来在公司把 ThunderBird 卸载再重新安装，就没有问题了，实在是 *^!\$%#^

有人用摩尔多瓦的 The Bat! 的么？没有特快专递，不支持 Yahoo! Mail、Hotmail，不支持新闻组，不支持一帐号多邮箱，不能够查看和编辑 HTML 源代码，不能显示外部链接图片…而且还不是免费的，除了 Educational License 之外，各版本价格在200-400元左右…好在现在的版本对 UTF-8 编码的支持要好些了…
在我看来就算打死也不用 Outlook，用 ThunderBird

也可以啊，不明白为什么许多人都说 The Bat!

Becky! Internet Mail 也是风评很好，但是一来又是收费的软件，二来那个界面实在太简陋了…

说来说去，目前还是 ThunderBird 最适合我，如果它占用的资源能够再少一点，功能再多一点－－说实话，它的插件真正实用的不多，而且更耗资源。

待解决的问题
邮件讨论

终于把侧栏精简到了现在这个样子，不过还没有完工，因为要多增加几个与 blog 关系不大的页面，然后再把侧栏上页面链接那一段也收起来。

需要增加 Recent Posts 和 Recent Comments，方便阅读者浏览。目前几个相关的插件好像都不能令人满意。

尝试通过 RSS Feed 导入 BlogSome 上的文章，但有以下几个问题：
1、即使文章所属的分类名称相同，导入后还是会新增分类。目前没有适合 WordPress 2.0 的批量修改分类的插件，比较棘手。不知道能不能在 WP-DBManager 这个插件身上打主意，虽然它能够使用 UPDATE 语句，但是我尝试执行 SELECT 语句不成功…不敢随随便便 UPDATE…
2、Post Slug 变成和标题名称相同的字符串，这样中文标题的文章，其地址就变成一串无法理解的符号。

要把 Quotes 给弄回来。因为之前使用的 Another Random Quote 插件每次显示的字符数有限，一些句子都是精简过以后录入的，所以这回换用 Random Quotes 之后干脆都录入完整版本的。

虽然用上了 Ultimate Tag Warrior 感觉很爽，但是还没有一个很安全又方便的方法给以前的文章都加上 Tag。

升级顺利完成
邮件讨论

如果你看到的这一篇没什么问题，就说明升级到 WordPress 基本完成了！真是少见的顺利呀，不愧是以方便安装和升级、插件丰富的 WordPress。

整个过程大概2个小时，其中大部分时间用来通过 FTP 工具删除、上传、覆盖文件了，如果你的站点 FTP 速度较快，升级过程花费的时间会少得多。另外，据说 WordPress 官方站点上给出的升级步骤遗漏了要删除 Cache 文件夹的说明，所以保险起见，大家升级时要多留个心眼。

剩下的工作就是补充几个插件和改造 Sidebar 了。说实话，我不爽这个过长的侧栏已经很久了，现在用了 TOR 之后，访问的速度快多了，正好来个大扫除。

准备升级 WordPress 2.0
邮件讨论

因为安装 Ultimate Tag Warrior 的时候受了打击，所以下定决心把“滞销书”一次升级到 WordPress 2.0 版本，如果这几天 blog 无法访问，说明正在升级，或者升级失败，或者恢复失败… Orz
另外 RSS Feed 也可能出现异常状况。

先在 blog 上记录一下步骤，到时候也好知道到底是怎么死的。

1、备份数据库。
使用 Database Backup 插件导出数据库表。没有用 PhpMyAdmin，因为我是个懒人。到时候出了问题再搬救兵好了。按照常理，像我这样经常很倒霉的人出问题的概率是相当大的，何况还要偷懒…

2、备份文件。
虽然一些 blogger 说 wp-content 目录中的内容可以不备份，但我修改过默认模板的文件，所以要备份一下，升级后再看具体情况作修改。

3、停用所有的插件。
目前使用的插件有 Adhesive、Another Random Quote、FancyTooltips、SearchNow、[Twilight

AutoSave](http://twilightuniverse.com/2005/04/twilight-autosave/)、[Ultimate

Warrior](http://www.neato.co.nz/ultimate-tag-warrior/)、[wp-Define](http://www.zerohex.org/)、[WordPress Database Backup](http://www.skippy.net/blog/plugins/)、[Better Feed](http://frenchfragfactory.net/ozh/my-projects/wordpress-plugin-better-feed-rss/)。

4、下载 WordPress 2.0 文件。
下载地址。

5、删除文件。
保留的目录、文件：wp-config.php、wp-content 目录、wp-images 目录，必要的话，保留 wp-includes/languages/ 目录、.htaccess 文件，以及其他一些自己修改过的文件。
删除剩下的 WordPress 1.5.x 的文件。

6、上传 WordPress 2.0 文件。
最后上传 wp-content 目录，覆盖的时候要小心谨慎。

7、执行 wp-admin/upgrade.php。
然后就看人品是否过硬了。

8、在 Options 面板中重新设置 Permalinks

9、更新、激活插件。
更新 Ultimate Tag Warrior、Better Feed、SearchNow，试用 Random Quote、ThemeSwitcher。

“理想的书店”小问答
邮件讨论

大头绿豆在豆瓣上问深圳的豆友“深圳有你理想中的书店吗？如果有，是哪儿？如果没有，那么你理想中的书店应该是怎样的？应该有哪些东西或者服务？”

0、你多久没逛书店了？如果很久，是因为什么？是和我一样，因为觉得深圳没有好的书店吗？

1、如果你常常逛，那末通常是因为什么？为什么不在网上买而要到书店去？

2、如果你在罗湖，有什么可以吸引你去一家南山的书店？坦白说，离得多远的书店是你能接受的？

3、你希望这家书店能有地方让你坐下来读书吗？顺便来点喝的？

4、如果它有一个足够的空间，你愿意和你的朋友们来这里聊天，或者只是听它的音乐吗？

5、如果你选择到这家书店发一个下午的呆，你能够接受的消费是多少？

6、你愿意把自己的一些旧书拿到这个书店卖吗，哪怕必须接受一些附加条件？

7、你愿意在这里买一些旧书吗，如果它确实讨你喜欢？

8、你希望在这家书店里看见一只猫或者什么别的动物吗？

9、你希望它是白色的吗？

在长沙的我也凑个热闹回答一下：
0、不算太久，不过比以前的频率低了。原因是工作的地方附近没有好书店，周末又一直很忙，又有不少书等着看完。

1、经常逛的原因当然是来瘾了，还有就是经常有让人感兴趣的新书到货。如果是问书店吸引我的地方，除了质量和价格，那就是书籍的口味、稀有书籍的数量、店主的服务态度、检索查询是否方便、阅读环境如何。不喜欢上网买书的原因嘛，我一定要在手里面捧着才算放心，看中了又不买太折磨人，况且买书也讲究氛围，而且说不定可以遇到有同好的美女 ;P

2、现在常去的书店离家里有好几公里吧？关键是要有直达的公交车，或者骑单车、寄存单车方便也行。

3、当然要有坐的地方，喝的倒不必，怕倒在书上不好交差－－可是好像经常一边吃饭一边看书哦，那好吧，咖啡、茶、果汁、牛奶都可以，但不要卖碳酸饮料。

4、聊天听音乐没兴趣，那已经是另一种氛围了。要是谁在我看到一本特别想翻翻看的书时硬拉着我聊天，我会发飙哦。

5、能不能够不消费啊？我的要求不算高，收费的服务对我来说是多余。

6、旧书？我会卖？那一定不是我的书…

7、当然要买。本地买不到，到外地去买都成。

8、无所谓，只要它不吃书不撕书就好，也不要太吵闹，否则必会招来人道毁灭－－我可不是无条件的动物保护主义者。有条件的话…长颈鹿怎么样？

9、是纯棉的吗？没有小熊或南瓜图案吧？(-_-#)
