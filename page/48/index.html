手机中的零号机
Table of contents

虽然我曾经专门写文表达了对手机中的打火机的强烈不满，但最近的一件事情使我的想法完完全全彻彻底底的改变了！它使我意识到，我错了，我真的错了，幼稚的我是不能够马上看出这样一款划时代的手机产品的真正实力的。

那是一个艳阳高照的日子，我把手机放在牛仔裤屁股后面的口袋中在公交车上痛苦的假寐，全然没有意识到人品界面学科上一个伟大的历史事件正在发生！当我到达目的地后掏出手机一看，是这样一个画面：

是的，当时就是这样…

后来我试着操作了一遍，发现至少也要以下步骤才能做到：
1、第一步解锁；
2、第二步解锁；
3、0键3次；
4、*键2次；
5、进入选项菜单；
6、选择保存号码；
7、选择新建号码；
8、选择联系人身份归类为家庭成员；
9、选择保存；
10、确认保存号码；

哎哟我的娘啊，杰克，这真是太神奇了！ 我操，小明同学，这真是忒牛逼了！
虽然这没有猴子在打字机上敲出莎士比亚著作那么夸张，但是也足够震撼人心了，毕竟主角只是屁股而已（屁股：喂，“毕竟…只是…而已”是什么意思啊？！）。

人品界面史上所有伟大的发明在这一刻被轰杀至渣！Xerox PARC、Apple/M\$/Google/Nokia/Motorola/SE… 以及一干人品界面设计大师在这一刻泪流满面！

你们能够开发出界面的易用性达到让屁股（还是半边）都能够独自打开键盘锁并保存联系人信息到号码簿的手机系统么？ ！！！！！ :worship:

Human-Computer Interface 中的 Human 一词将被 Hip 代替，一门新的学科就此诞生！

给家里的电脑也装上 Chrome 试试不干正事的浏览体验，结果…

遇到 FeedBurner 这类的地址时，我会格外想念 FoxyProxy；
没有 Autofill Forms 和 InformEnter 的协助，填写表单变成一种折磨；
快速复制不能少了 Auto Context，强力复制又不能少了 CoLT；
拖拽用的不多，但真正要用的时候，还是有个 Easy DragToGo 比较顺手；
便捷直观不如 Download Statusbar，内功修为不及 DownThemAll!，下载我还用不惯；
虽然不用 AdBlock Plus，但至少也该有 Flashblock 吧？
不支持 Greasemonkey 这样的脚本平台就少了很多精彩…
看到超大图片总是会提醒我该使用 Image Zoom 了；
Auto Dial + Read it Later + Taboo 才是历史页面备忘管理的王道；
如果 Flickr 再被墙奸，就能体会到 Access Flickr! 是多么的有爱了！
…

所以最终的结论是…

Flickr Pro
Table of contents

最近终于购买了 Flickr Pro 的服务。其他的相册服务虽然各有特色，但是总的来说，即使付费，Flickr 也是当仁不让的首选！

于是每天都在大量的上传积存的照片（所以不能保证质量）。虽然整理大量照片不如本地方便，但 Flickr 仍然是我使用过的相册服务中在这方面最强的。诀窍是在 Organize 中活用照片集、筛选、搜索和“Jump to date”。

为了网上付款的事情，在多次被建设银行调戏之后，一怒之下把钱都取出来全部存到了招商银行，两边的服务一对比，才觉得自己像个真正的顾客了。

看到 Twitter 上有人提起 PSPad，看看新版本有何变化，发现功能增强了（尤其是脚本功能！还有一堆新增的贴心小功能），对 Unicode 的支持也好很多，险些要用来当作默认文本编辑器。但又发现字体只支持等宽字体，中文很难看不说，一堆 Unicode 字符也不能用了，看来仅适合写代码人士使用了…

今天的大消息是 Google Chrome（谷铬？）即将现身江湖，重复的话就不多说了，只说两点：
1、从描述和截图来看，Chrome = Safari+Firefox+Opera+IE8+Prism？
2、Chrome 的重点恐怕在于移动客户端市场，桌面上最受威胁的应该是 Safari Windows 版。

第二代身份证还没有到手，打电话问了一下公安局，答复说新身份证要在办理后60天才能拿到。那开始白纸黑字写30天之内是什么意思？还有人说等了8个月才拿到，只怕到时候我的联系方式都变了…

凌晨2点，Kahn 告别赛，Bayern 对德国队。今天才刚回味了那个 Kahn 还在 Karlsruhe 时被 Okocha 戏耍的视频…
现在很多人说02年的 Kahn 如何厉害，其实98年他就应该替代老迈的 Kopke 当正选门将了，可惜他的巅峰期只留给国内联赛和冠军杯。

OM 的正太 Samir Nasri 去了 Arsenal 之后表现还不错，不少曾经认为他是水货的枪迷已经改变了看法。我倒觉得现在还不能说正太已经较快的适应英超和 Arsenal 了，等这个赛季踢完了再看看总体评价吧。另一方面，正太的控球基本功很扎实，分球的头脑很清晰，加上潜力还有上升空间，如果不把他限制在 Pires 接班人的定位上的话，假以时日，他是能够超越 Hleb 甚至 Pires 的成就的——当然，如果 Arsenal 用不好他，那么还是回我们 OM 吧，免得暴殄天物啊～
