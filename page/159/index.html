My DoNews
Table of contents

DoNews 推出基于 WordPress 的 blog 服务站点：My DoNews，刘韧声称“这是全内容的Blog”。它的前身估计就是让许多人意见很大的 WordPress.com.cn。

注册了一个账号试试，很自然地不如预期的那么好。

1、注册时提示我 blog 地址为 username.my.donews.com，但是注册之后又告诉我，blog 地址是 my.donews.com/username。我倒不特别在乎 blog 域名的长短，只要不是新浪那样的数字 ID 就成，可真正在乎这种事情的用户会怎么想？才翻过了一个页面而已，就口径不统一了？
2、和 WordPress.com 一样，无法由用户管理插件。我希望至少能像 BlogThing 或者 Blogs.ie 一样开放 CSS 编辑功能吧，否则我怎么解决 ChinaRed 主题中文字体太小的问题呢？而且 My DoNews 上使用的 ChinaRed 主题还有偏左的问题。
3、自己搭建 WordPress blog 或者去国外 BSP 注册，界面全英文没有关系。但是为国内大众服务，全中文界面就很有必要了。而现在 My DoNews 还是 WordPress 2.0 的英文版本。这个已经有不少人在抱怨了。
4、无论是 blog 展示页面、注册界面还是管理界面，都没有提供到用户论坛的链接，甚至到 DoNews 本身的链接，如果服务出现问题，不了解 DoNews 的用户该怎么办，到哪里去反映情况、提出建议和意见？在 DoNews 社区中好像也没有专版负责这项工作。或者是我没有找到？那也说明链接不够醒目…

《金刚》确实是一部不错的翻拍电影，虽然最后在帝国大厦那一段我早有预见而没有太多触动，但海岛边金刚近距离遭氯仿袭击后，用尽最后的力气摊开手掌伸出去的那一幕－－还配合女主人公 Ann 的哽咽－－确实挺有杀伤力的。

其它方面么，几段无声画面的处理很到位，对岛上怪异生物描述的份量过多，轮船缩微模型、金刚奔跑时握人的手破绽比较明显，至于金刚的身材比例、动作幅度与骨骼承受能力的关系、被省略的运输大猩猩过程、女主人公下雪天穿那么少还在大街上狂奔在帝国大厦塔顶看风景等等，那就是柳田理科雄《空想科学读本》的事情了 XD

还有，男主人公 Jack 真的很惨呀，因为 Ann 亲眼目睹金刚为保护她与恐龙搏斗，留下了非常深刻的印象，对他的冒险经历倒缺少关心，出生入死后得到的关注还不如大猩猩，回到纽约后也就是最后在 Ann 面前非常“及时”地露了一小脸…小心以后会被发好人卡啊。

最后，冒昧地问一句，这算是《Jurassic Park》外传么？(-_-#)

为什么要看康德的书？
Table of contents

配合置顶提醒贴而写。

1、因为康德是个聪明的人。
交朋友可以不看智商，娶老婆可以不看智商，看书却一定要先看看作者的智商。

2、因为康德是个喜欢怀疑、不被权威意见左右的人。
“我们的时代是真正的批判时代，一切都必须经受批判。通常，宗教凭借其神圣性，而立法凭借其权威，想要逃脱批判。但这样一来，它们就激起了对自身的正当的怀疑，并无法要求别人不加伪饰的敬重，理性只会把这种敬重给予那些经受得住它的自由而公开的检验的事物。”

3、因为康德的哲学体系是现代哲学研究和科学哲学研究绕不过去的重要体系。
从黑格尔到马克思、从叔本华到尼采、古典唯心主义到马克思唯物主义到实证主义到当代科学哲学等等，这个展开来就太啰嗦了。

4、因为三大批判从起步阶段就是在脚踏实地的基础上，为彻底地回答哲学终极问题而构建体系的。
用我今后将会经常提到的一个比喻来作说明就是，人类像是井底之蛙一般，在无知的幕布包裹下努力探寻道路，企图跳出井外完成“认识我自己”和“认识世界”的无尽任务，而康德的哲学体系打算从人最基本最纯粹的知性、判断力、理性能力入手，为这只青蛙的攀爬跳跃构筑坚实的基础。它不求能够看到最终完成这个不可能任务的那一天，至少保证已经构建的体系能够被后来者放心大胆地当作巨人的肩膀。
从这一点上说，康德的想法就比那些急于求成、期望一步登天、立即解答终极问题的哲学家和宗教信徒要高明得多。井底之蛙可笑之处不在于它所处的困境，而在于它还没有爬到井口外面就开始信口开河了。

5、因为半吊子自由主义者需要回到康德的思想中寻找自由主义之根。
承莱布尼茨、牛顿、休谟、卢梭等人的思想又对其进行批判，然后自成一代宗师的康德，其哲学体系对其后的自由主义的影响是深远的，特别地体现在他对“自由意志”和“道德律”的论述上。由于他夯实基础的努力，从他的哲学体系中寻找自由主义之根，比跟着那些宪政基督徒、自由主义基督徒、保守主义者等等一起囫囵吞枣，恐怕要有希望一些。

6、因为他是光荣的去死去死团员。

中长期任务
1、把《康德三大批判精粹》看完，用自己的语言总结归纳出来。结合相关的书籍进行深入学习。去年没完成任务，今年继续，今年完不成，明年继续。
2、在第一步完成的基础上，总结对科学的认识，写成系列的科学哲学文章。这个计划也是在《智慧设计论的争论》一文中提到过的。名字？或许叫做《奥卡姆剃刀教宣传手册》吧？(-_-#)

KooMail 或 ThunderBird
Table of contents
    • 也可以啊，不明白为什么许多人都说 The Bat!

今天把公司电脑上的 ThunderBird 用覆盖旧版本方式升级到了 1.5，结果安装完毕之后竟然无法启动，重启之后依然没有解决问题。按某人的话来描述就是：

We uninstalled everything and reinstalled fresh. No go. We rebooted. No go. We uninstalled, rebooted and reinstalled. No go.
When one double clicks the icon, nothing happens. Task manager shows nothing. No windows. No event logs. No pop-ups. No sounds. No chicken nuggets. Nada.
用户体验可谓极差，安装之前也不打一声招呼。
于是临时找来 KooMail 作替代品。

KooMail 占用的资源不算太多，响应比较快，据说 DreamMail 的响应速度就很成问题，程序开得多的话，我的 ThunderBird 也经常反应迟钝。

远程管理的功能不错，可以在服务器上设置废件箱、草稿邮件箱、已发邮件箱。不知道其他客户端能不能够做到。

其他的功能嘛，和 FoxMail 和 DreamMail 都大同小异。自动分割大邮件、分析 IP 地址、建立虚拟子邮件夹的功能不错，但并不新鲜，而且无法建立多级虚拟子邮件夹，听说同时应用多个搜索条件也存在问题。

RSS 功能比较鸡肋，手工删除安装目录下的 RSS 目录或者在查看中禁掉即可。

过滤垃圾邮件的功能比 ThunderBird 差远了，和 FoxMail 比也不是一个档次。

看到作者挺乐于接受用户的建议的，于是去论坛留了一大段话，希望能够改进。

当然，后来在公司把 ThunderBird 卸载再重新安装，就没有问题了，实在是 *^!\$%#^

有人用摩尔多瓦的 The Bat! 的么？没有特快专递，不支持 Yahoo! Mail、Hotmail，不支持新闻组，不支持一帐号多邮箱，不能够查看和编辑 HTML 源代码，不能显示外部链接图片…而且还不是免费的，除了 Educational License 之外，各版本价格在200-400元左右…好在现在的版本对 UTF-8 编码的支持要好些了…
在我看来就算打死也不用 Outlook，用 ThunderBird

也可以啊，不明白为什么许多人都说 The Bat!

Becky! Internet Mail 也是风评很好，但是一来又是收费的软件，二来那个界面实在太简陋了…

说来说去，目前还是 ThunderBird 最适合我，如果它占用的资源能够再少一点，功能再多一点－－说实话，它的插件真正实用的不多，而且更耗资源。
