限制老人高峰期挤公交？
Table of contents

城市公共交通压力不断增加，老年人比例上升，加上一直实行的凭老年证免费乘车制度，近几年许多在上下班高峰时期挤车的上班族提出：老年人能否避开上下班高峰期乘车，更进一步的，还有政协委员建议限制老人高峰期挤公交。

作为一个每天挤本城最拥挤公交（之一）上下班的死上班族，我觉得这样的建议相当无理。

凭什么断定老年人的出行没有年轻人上班重要？等到你老了的时候会不会觉得相对于为别人卖命卖青春，还是清早出去锻炼身体更有意义？何况还有帮子女送小孩上下学的老人，倒遭人嫌了。
如果老年人出行避开上下班高峰成为了社会的共识，那么那些真正有急事出门的老人会不会也成为人们鄙视的对象？莫非要他们先解释自己出门是有正当理由的？
给老年人让座应该是提倡的道德行为，并非不得不做的事情，假如因此对老年人心怀不满，又何必装睡，干脆心安理得的不起身让座好了。
如果说高峰期挤公交对老年人有许多坏处，多多宣传就行了，也犯不着打着“为你好”的牌子限制他人。
最根本的问题是，公共交通拥挤的问题应该是由市政部门来解决吧？为什么第一反应要牺牲老年人自由出门的权利呢？无论是普通市民，还是政协委员，仿佛天然的站在本该做好公共交通服务的公共客运行政主管部门和公交公司的立场，可能是习惯成自然了吧。

另一个看不惯的现象是，提着大包小包上车的外地乘客（通常是在几个汽车站之间转车的民工）有时会被司机和其他乘客指责占了位置、影响他人，遭遇集体白眼攻击，最厉害的时候，司机还会拒载，然后其他乘客说“拿着这么多东西就该去打的”。
提醒人家挪挪货物的位置，按公交规定收取货票钱不就可以了吗，何必赶尽杀绝！
虽然现在民工挣得比本人还多，但多打几次的，辛苦挣来的钱还不都进了的士司机的口袋了？莫非买不起私家车的“搬运工”都是活该遭人白眼的失败者？
需要别人体谅自己挣钱辛苦的上班族，这个时候也该体谅一下别人吧。

Chrome 扩展
Table of contents

首先到这个页面下载 Chrome 4.0 Beta，然后去扩展页面淘吧。

Chrome 扩展的好处：
不需要重启就可以生效和卸载；
扩展有独立进程。

不佳之处则有：
官方扩展页面无分类；
浏览器内的扩展管理页面找不到扩展的主页；
使用其他浏览器时不能先下载再安装。

等到 Chrome 扩展的丰富程度超过 Firefox，而性能仍然能够保持最快的水准，我就会更换默认浏览器了。

目前我已经安装的扩展有：

Diigo bookmarks
书签收藏类，可惜 Diigo 至今仍然处于被封杀状态。

FlashBlock
阻挡 Flash 就可以节省很多资源了。

LastPass
密码管理。

Lazarus
自动保存输入框内容，防止丢失未提交的文本。

Session Manager
存储、还原会话。

Switchy!
多代理切换。

VerticalTabs
标签太多的时候可以以纵列形式查看标签标题文字。

不少 Firefox 的扩展要继续耐心等待才能在 Chrome 上看到，有的也许永远也看不到…
