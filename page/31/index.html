限制老人高峰期挤公交？
Table of contents

城市公共交通压力不断增加，老年人比例上升，加上一直实行的凭老年证免费乘车制度，近几年许多在上下班高峰时期挤车的上班族提出：老年人能否避开上下班高峰期乘车，更进一步的，还有政协委员建议限制老人高峰期挤公交。

作为一个每天挤本城最拥挤公交（之一）上下班的死上班族，我觉得这样的建议相当无理。

凭什么断定老年人的出行没有年轻人上班重要？等到你老了的时候会不会觉得相对于为别人卖命卖青春，还是清早出去锻炼身体更有意义？何况还有帮子女送小孩上下学的老人，倒遭人嫌了。
如果老年人出行避开上下班高峰成为了社会的共识，那么那些真正有急事出门的老人会不会也成为人们鄙视的对象？莫非要他们先解释自己出门是有正当理由的？
给老年人让座应该是提倡的道德行为，并非不得不做的事情，假如因此对老年人心怀不满，又何必装睡，干脆心安理得的不起身让座好了。
如果说高峰期挤公交对老年人有许多坏处，多多宣传就行了，也犯不着打着“为你好”的牌子限制他人。
最根本的问题是，公共交通拥挤的问题应该是由市政部门来解决吧？为什么第一反应要牺牲老年人自由出门的权利呢？无论是普通市民，还是政协委员，仿佛天然的站在本该做好公共交通服务的公共客运行政主管部门和公交公司的立场，可能是习惯成自然了吧。

另一个看不惯的现象是，提着大包小包上车的外地乘客（通常是在几个汽车站之间转车的民工）有时会被司机和其他乘客指责占了位置、影响他人，遭遇集体白眼攻击，最厉害的时候，司机还会拒载，然后其他乘客说“拿着这么多东西就该去打的”。
提醒人家挪挪货物的位置，按公交规定收取货票钱不就可以了吗，何必赶尽杀绝！
虽然现在民工挣得比本人还多，但多打几次的，辛苦挣来的钱还不都进了的士司机的口袋了？莫非买不起私家车的“搬运工”都是活该遭人白眼的失败者？
需要别人体谅自己挣钱辛苦的上班族，这个时候也该体谅一下别人吧。

Chrome 扩展
Table of contents

首先到这个页面下载 Chrome 4.0 Beta，然后去扩展页面淘吧。

Chrome 扩展的好处：
不需要重启就可以生效和卸载；
扩展有独立进程。

不佳之处则有：
官方扩展页面无分类；
浏览器内的扩展管理页面找不到扩展的主页；
使用其他浏览器时不能先下载再安装。

等到 Chrome 扩展的丰富程度超过 Firefox，而性能仍然能够保持最快的水准，我就会更换默认浏览器了。

目前我已经安装的扩展有：

Diigo bookmarks
书签收藏类，可惜 Diigo 至今仍然处于被封杀状态。

FlashBlock
阻挡 Flash 就可以节省很多资源了。

LastPass
密码管理。

Lazarus
自动保存输入框内容，防止丢失未提交的文本。

Session Manager
存储、还原会话。

Switchy!
多代理切换。

VerticalTabs
标签太多的时候可以以纵列形式查看标签标题文字。

不少 Firefox 的扩展要继续耐心等待才能在 Chrome 上看到，有的也许永远也看不到…

QQ 秀是不是在骗钱？
Table of contents

QQ 秀是不是在骗钱？
按照 Datou 的说法，QQ 秀是明码标价，不存在欺骗行为。

但是，为什么我一打开 QQ 聊天窗口，我自己的布衣 QQ 秀就一定要出现？
并且只能暂时收起侧栏，找不到今后不再显示侧栏的设置。下次打开聊天窗口 QQ 秀仍然会出现。

即使彻底注销掉 QQ 秀，也会有空白的鬼影形象占用侧栏空间。

而且还是不能永久取消，至少在我的 QQ 2009 Beta (375) 版本上和官方页面找不到设置的地方。腾讯官方的答复也是无法永久取消。

免费赠送加多了盐的面包，然后在旁边摆摊卖矿泉水，这种行为总觉得哪里不对劲啊。
我想，“体验经济”不应该包括“必须体验”的体验吧，至少当我不想体验的时候不要来烦我。

爱足球还是虚荣
Table of contents

作为铁杆法国队球迷，此次深以法国队附加赛第二场的晋级过程为耻。除了神勇的 Lloris 和差强人意的 Anelka，这场比赛的法国队可以说是一无是处，小组赛后段的复苏被打回原形，06世界杯之后的痼疾依然还是痼疾，未成形的战术仍未成形，依我看不如被淘汰出世界杯之外，让多没内裤早些走人，安心完成新老交替换血和新战术体系的重建吧。

我不知道某些号称是铁杆球迷的人，爱的究竟是足球还是只是虚荣。
为了给自己喜爱的球队和球员说好话，他们可以公然称赞暴力、作假，将竞技体育公平竞赛的原则视若无物，而只愿意接受竞技体育的荣誉。

无论是北京国安夺冠过程中的闹剧，Evans 恶意飞踹 Drogba，还是 Henry 故意手球停球助攻帮助法国淘汰爱尔兰，都可以看到护短的球迷无基本的是非观念，维护甚至称赞本应是耻辱的行为。

假如到处都充斥着这种势利的球迷，假如足球彻底沦为为达目的不择手段的运动，足球作为一种“game”还有什么乐趣可言呢？假如所谓的球迷将足球运动本身的乐趣排在第二位，只看重最后的结果，大家还是打开 FM 直接调出球队或球员的历史荣誉或者靠数钱来满足自我算了。
