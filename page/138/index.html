本游戏和求知无关
Table of contents

下午嘻嘻 TV 10频道的《探索·发现》栏目在说纳粹德国的事情，谈到亚特兰帝斯、西藏的地球轴心、飞碟、希特勒未死、空心地球等等事情，主持人说到最多的词是：“据说”、“他说”、“他声称”、“有报道说”…特别是“据说”一词，几乎每段话都要带一个。但是是谁说的？在何时说的？有没有可检验的记录？有没有提供有力的证据？他们根本不关心，因为他们的目的只在利用观众的好奇心，至于误导观众的责任，他们是没想过也没打算要承担的。

为什么要对观众负责呢？科普/教育频道不也可以振振有词地如此说吗：你怎么就知道这些故事是假的呢？既然你无法绝对地否定其真实的可能性，那么拿来说一说赚一赚眼球有何不可？所以，即使提到第二次世界大战期间据说纳粹德国有可能差一点研制出时速达到3200公里/小时的碟形飞行器，主持人还要作好学不倦状说，“真实的情形到底如何呢？恐怕永远是无法解开的谜底了。”我敢说如果他去法庭当辩护律师，所有的被告便都有救了。
另一方面，既然观众都喜欢这种无知带来的廉价乐趣，和太容易满足的求知欲（大概只是装作要求知而已，因为他们不欢迎那些不符合自己期望的真实），那么为什么要对提供给他们的信息负责呢？说些不痛不痒、谁也不会真正关心真假的胡话，吸引眼球，拉动内需，打发无聊，娱乐大众，降降智商，赚赚钞票，大家受益，其乐融融，岂不妙哉？
好吧，那么所谓的探索和发现呢？得了吧，除了“等待 XX 政府解密绝密档案”之外，你们还能干啥？

升级到 WordPress 2.0.3
Table of contents

从 WordPress 2.0 升级到了2.0.3稳定版。这一次偷懒没有先停止插件，结果升级后文章无法显示，报如下错误：

WordPress database error: [You have an error in your SQL syntax; check
the manual that corresponds to your MySQL server version for the right
syntax to use near ‘AND comment_approved != ‘spam’ ORDER BY
comment_date’ at line 1]
SELECT * FROM wp_comments WHERE comment_post_ID = AND comment_approved
!= ‘spam’ ORDER BY comment_date
想来想去应该只有可能是插件不兼容造成的，只好乖乖地用枚举法逐个停掉，最终定位到 Adhesive 上面。
所以以后升级还是不要偷懒的好，先停插件不是坏事。

买旧书
Table of contents

下午流窜到福州路，终于找到了2年前买《艺术哲学》的旧书店，具体地址是福州路401号，但是是邻近415号的，我也不知道为什么。店名是上海图书公司，同时写了上海社科书店的名字，旧书店在4楼，3楼也有一部分。从上海书城正门出来向东走一小段，街道南面。懒得开 Google Earth 了。

今天有部分摊位没开，剩下的当中合口味的书不多，特别是古老的科普书不多，2年前中意的似乎都消失了，看来只能期待下一次再来上海时能有意外收获。

买了4本书：
茨威格的《三大师：巴尔扎克、狄更斯、妥斯妥耶夫斯基传》，标价3元，1元1个大师，挺划算；
麦德维杰夫《让历史来审判－－斯大林主义的起源及其后果》，一共2本，加上1本续篇，3本标价40。
总共30元拿下。

敏思博客停止运行
Table of contents

敏思博客因为财政困难停止运营，预计8月1日关张，现在已经开始通知用户备份文章，另寻新家。因为资金问题而玩不下去的 BSP，我看到的这还是第一家。

国内的 BSP 本来就有可能玩到半路没兴致了甩手不干，可能被警察蜀黍搬走服务器，也可能不打招呼就删档，现在又来一个财政问题…所以说，既然命运无法捉摸，选择 BSP 就要留个心眼，看看是不是方便导出存档，并且易于转移到其他 blog 平台上，因此选择使用主流 blog 程序的 BSP 是相对明智的，至少不用导出数据之后还要等人帮忙作格式转换。
