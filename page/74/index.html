Firefox 3 Beta 1
Table of contents

本来打算等到 Firefox 3正式版本发布之后再用的，但实在经不起诱惑，还是下载了 Beta 1版。

首先，最大的感受就是速度确实是上去了，资源占用少了。是不是因为大部分扩展都失效的缘故？也许是原因之一，但同样是裸奔，还是感觉比2的光猪跑得快。

其次，目前大部分的扩展还没有跟上 Beta 的进度，要么选择裸奔，要么自己手动修改，要么选择使用 Nightly Tester Tools 扩展。
我有39个失效的扩展和主题，仅有2个存活。有的没了忍一忍也就算了，但是像 CoLT、TwitterFox、Greasemonkey、Diigo、Del.icio.us、Access Flickr!、Flashblock、Image Zoom、InFormEnter、It’s All Text! 这些扩展无法用还真是像断手折足一样惨啊…

西行资讯翻译了一下新增特性，我仅说一下一些自己看到的地方：

1、在陌生网站安装扩展的时候，不是跳出白名单列表，而是直接选择是否允许安装：

2、插件管理和扩展、主题放置在了一起，而且不必非得安装附加软件之后才能重启：

3、地址栏中的自动补全貌似采用更模糊的匹配模式：
Firefox 2的地址栏自动补全：

Firefox 3 Beta 1的地址栏自动补全：

4、退出时会询问是否保存当前的标签页并在下次启动时打开：

5、在地址栏右侧新增书签星标按钮，点击一下是将当前页面加入书签，再点击则是编辑书签：

但我觉得书签的管理界面很混乱。上面的快速添加界面可以编辑标签，却不能够编辑关键字，要编辑关键字还得在书签菜单中查看书签属性时完成，可问题是，书签属性界面却没有编辑标签的地方…

如果想要在同一批次同时编辑书签的标签和关键字该怎么办？那只好打开书签管理器，或者用 Firefox 3的新名词，Places 管理器：

对我这个很少打开书签管理器，希望在添加书签时就搞定一切的用户来说，这样的设计实在很糟糕。尤其是 Places 的新特性可能吸引我更多地使用本地书签的时候。

6、在书签菜单中根据多种条件展示书签和历史页面的 Places 应用：

7、登录网站时，变成在页面上方出现提示条，询问保存密码的方式：

但原本不是3个选项吗？暂时不记录的选项难道又消失了？ :050:

8、点击网站的 Favicon，会出现网站安全信息：

9、貌似 title 文字显示的长度和时间都有增加，可能不再需要 Long Titles 扩展了。

暂时只发现一个不太稳定的现象，非当前标签页的关闭按钮在鼠标移上去时可能无法激活。

像中国队一样去战斗！

以色列人主场战胜俄罗斯，奇迹般的将英格兰人从地狱拉回了天堂，将欧锦赛预赛出线权放进了英格兰人的口袋里面。
但是今天早上，在打平就可以出线的情况下，英格兰人自己又把出线的希望抛掉，从云端倒栽下来，面部着地。
“恭喜英格兰队，成功避开了所有欧洲强敌！”
“这一刻，中国队灵魂附体！”

Robinson 靠不住，但缺少经验的 Carson 在如此重大的比赛中更加靠不住（同样的道理，缺少国家队经验的 Frey 第一次首发也犯下了低级错误）。
当然，门将低水准发挥的背后是全队表现差劲，被已经出线的克罗地亚开场就摁在地上打（开场前走客队国歌时没素质的英格兰球迷狂嘘，显然很伤人品）；
全队被海扁的背后是主教练排兵布阵上自废武功；
而主教练平庸无能的背后则是英足总选帅时气走小组出线直接竞争对手的主教练 Hiddink，以及选人时的昏庸保守；
再往下面说，恐怕就要扯到教练培养、青训、技战术思想了…

16支决赛阶段球队全部产生之后，再看看分档情况：
种子：瑞士、奥地利、希腊、荷兰
二档：克罗地亚、捷克、意大利、瑞典
三档：罗马尼亚、葡萄牙、德国、西班牙
四档：波兰、法国、俄罗斯、土耳其

分档规则是根据2006年世界杯预选赛和本届欧洲杯预选赛的战绩来划分，感觉非常不合理，世界冠军去二档，世界季军和殿军兼上届亚军去三档，世界亚军去四档，而在近两次大赛中被三档球队葡萄牙淘汰的荷兰却位列一档…

加上两位东道主占据种子球队席位，那么会不会产生史上最强的欧洲杯死亡之组呢？

荷兰/希腊/瑞士、意大利、德国、法国

猴子的故事
Table of contents

昨天从《基督教世界科学与神学论战史》中摘录了一则故事发到 Twitter 上：

从前，有一个快乐的法国国王在他的王国中巡游视察，他在一个省城的城门口受到了市长代表的欢迎，这位代表这样开始了他的致辞：“祝您快乐，陛下，今天上午市长大人不能来恭候您的大驾光临，理由一共有13个。第一个理由是他死了。”听到这里，国王仁慈地宣布，这第一个理由已经很充分了，他不想再麻烦市长代表继续讲另外12个理由。
结果 Vanny 同学要求我编个新的故事，我一不留神就答应了，下面是这个刚刚编的故事：

从前有一只猴子，它和几十个同伴一同生活在丛林中。
除了不得不一起去摘桃子，这只猴子大部分时间都是无所事事的，特别喜欢若有所思的发呆，或者就是抬头看看天上的太阳呀月亮呀星星呀云朵呀一看就是半天，要么爬到树上往下面扔大大小小的苹果，要么对着树洞吱吱唔唔说些什么同伴都听不懂的话。
某一天，它宣称自己发明了数学——虽然它的同伴都不懂这是什么，但它很快通过这个新发明向猴子头证明最近一个月以来桃子的产量确确实实是下降了的，而且给出了精确到小数点后两位的计算值——猴子头暂时还没有理解百分比是一个什么概念，但它很高兴其对桃子收入减少的担忧得到了某种高级理论的支持，于是在猴子头的支持下，那只猴子得以完全什么事情都不干，并教授其他的猴子学习数学。
猴子们从此幸福快乐的生活着，它们享用着那只猴子陆续发明的文字、发电机、移动电话、身份证以及剃须刀，并建立了跨国公司和股票交易市场，直到某一天那只猴子发狂后从自己的别墅跳下来摔死了。
其他的猴子都不明白它为什么要自杀，在检查了尸体之后它们继续去树林里到处刻“办证 13xxxxxxxxx”的字样。

为什么现在即兴编故事的水准比小学时候差了很多啊～ :036:
