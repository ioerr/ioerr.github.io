快餐印象
Table of contents

以前很少去肯德基、麦当劳吃饭，主要是性价比不高。后来经常出门乱逛，倒是喜欢去那里面用餐。不是本地的杨裕兴、甘长顺、黄春和以及各种小吃店不好吃，而是相对而言环境更好（窗明几净灯光好适合看书）、方便上厕所，以及质量稳定，不容易吃到一顿糟糕的饭菜。

最近又去了几次，发现有些以前没看到的变化：
1、有家麦当劳把所有食物的价目表贴在了服务台旁边的墙上，这样视力不太好的人就不用挤到队伍前面去看服务员背后的价目表。我记得之前不是这样的，只有促销的套餐说明才放在服务台上。
2、因为排队的人太多，所以有服务员跑到队伍中间询问点餐内容，然后写在纸条上交到顾客手里，这样就减少了顾客和收银员沟通的时间。
3、如果某种食物暂时供应不上，就打个“白条”给顾客，过一段时间再来自取。很久以前有过一次服务员陪我站在那里一同等某个汉堡，用时刚好5分钟。
4、如果吃到一半想要加点，又怕座位被别人抢了，可以请服务员帮忙。
5、端着餐盘半天找不到就座的地方，服务员会主动询问其他顾客旁边的座位可不可以坐（其结果是残酷的粉碎了我凑到美女旁边坐下的企图，悲剧啊～）。

可见，人太多才是根本矛盾啊…

今天出门吃的是德克士（Dicos），照烧鸡排饭套餐。之所以脑子进水去那里吃，一是因为肚子很饿，二是因为以前没体验过，想要试试。
不试则已，一试惊人，鸡排像微波炉加热的冷冻食品，鸡翅又咸又辣，配汤没什么咸味，其中的紫菜却有腥味，总之我的人生又多了一道阴影…

印象中大概10年之前，先锋厅那里开过一家牛肉汉堡快餐店，老板颇有雄心壮志，舍得下血本，牛肉号称都是美国进口的，味道确实不错，从美国过来的两个表弟什么都吃不惯，就只吃它的汉堡。当然，价钱也相应的不便宜。后来半年之内就倒了…

EULAlyzer
Table of contents

EULAlyzer 的想法是不错。最终用户许可协议（End User License Agreements，EULA）普遍太长太繁琐，导致无论用户有没有兴趣都读不完，于是它就帮用户自动分析一下协议文本，精炼出最重要的信息。

但我总觉得这种问题要靠第三方软件解决，实在是开发者之耻，何况 EULAlyzer 也不是万能的分析器，不能对付所有的语言和千奇百怪、要点各不相同的协议，不同的用户阅读同一个协议的时候，关注点也不尽相同。

那么为什么不可以有一个最终用户许可协议的文本规范呢？而且像提供软件安装工具一样提供最终用户许可协议打包和展示工具，或者干脆集成也好，最起码都写在规范统一的 XML 文件中，告诉像 EULAlyzer 这样的工具如何自动分析。
个人发布软件的时候，在别人的最终用户许可协议基础上修改也会省事吧…

手机中的打火机
Table of contents

本来很有冲动为手头的这个 PHS 山寨机专门写篇文章，让大家看看其惨绝人寰的功能和人机界面，但越是折腾就越是心寒，最终我对全面剖析它已经绝望了…

以下仅列举出几方面：

1、号称音乐手机，面板上书 Music Phone，包装盒号称“64和弦/真人真唱”，还有不规范的专用 USB 接口。
结果呢？USB 口仅能用于充电，电脑无法识别，导出/导入数据/音乐就不用想了。
这也罢了，称的上是音乐的铃声只有茉莉花、笔记、山路十八弯、好姑娘、你到底爱不爱我，其中前两首还是 MIDI…
我是不是应该庆幸它根本就没有提供音乐播放器？

2、输入法虽然比较齐整，但是无法调整默认输入法，也就是说，即使在同一个窗口中填写不同条目的内容，只要是要求输入中文的，一切换过去永远都是拼音输入法出来…
后来在输入联系人名单的时候，我终于意识到，这是在暗示我最好在现实中把联系人统统都干掉以便精简操作。

3、你相信吗？设定每日闹钟时，你能够选择的只有闹铃的时间和铃音，而且只有一组闹铃。

懒人模式重复响铃？你够本找不到开关在哪里！更不用说调整间隔时间和响铃次数了。
于是早上的时候每隔一分钟（！）我就被闹铃骚扰一次，而且至今不知如果不手动终止，它会响铃到什么时候…

4、玩游戏的时候，按下右软键是返回上级菜单。可是它既不会询问你是否真的要退出，也不会为你保留游戏状态，直接就中断了，返回之后一切归零，成功的营造出了一种恍然如隔世的感觉。

5、增值服务中有腾讯 QQ 之类的子项目，如缘分交友、快捷菜单等等，但是无任何说明，恕我鲁钝，我无法光从名字就理解其全部服务的深刻内涵。
尝试着点击，出来的信息只有“提交成功！”
可但是，但可是，
提交了什么？
提交给谁了？
提交了又怎样？
为什么要提交？
这项服务究竟是什么东西？
没有人回答你。
除了几分钟后发过来的一条要你确认交钱的短信…在对你的理解能力进行嘲笑之后最终来一个附加伤害。
如果是在移动公司没有整顿 SP 之前这样操作，我会不会亏死？

6、电话簿群组管理…
对不起，我到现在还没有完全参透电话簿群组管理的奥义，它让我明白了什么叫做“天外有天”。

7、很让人意外的，在 PHS 终端最有竞争力的“电池续航能力”方面，它也是软蛋一个，2天时间就变成只有1格电…是我见过的电池最差的 PHS 终端…

最后是外观，看上去还是不错的，轻便小巧：

生产商是深圳天耀科技有限公司，型号是 F35E，可能因为是限地区销售，这款型号的机器并没有出现在它的产品清单上。

不要问我为什么手上会有这一款机器，这当然不是我选的。你们可以不相信我的人品，但一定要相信我的品位！
