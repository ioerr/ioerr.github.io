Dash Command
Table of contents

Life Hacker 终于想起来要介绍 Executor 这个功能强大的快速启动工具，留言中的群众纷纷表示 Executor 很好很强大；同时，因为它的设置选项太丰富，能够挖掘的东西太多，也吓走不少喜欢简单快速上手的 Launchy 用户，而且，有的用户并没有深入研究就认为 Executor 具有某种缺点：比如，以为一定要先用箭头向下移动才能选择第一候选，以为 Launchy 使用 runner 插件添加别名就能达到和它一样效果。

额外的收获是，看到有人推荐自己参与开发的另一款快速启动工具：Dash Command（急急如勒令？命令大师？囧）

Dash 是一款收费软件，目前还没有到1.0正式版，预售优惠价19.95，正式售价\$50，与其他免费好用的同类软件相比这是个很大的劣势。
但它的优势是系统资源占用相对小很多，同时又保持了漂亮的界面和强大的功能。

下面是试用版的使用体验：

首先，基本的使用方法都差不多，呼叫出控制台，敲入关键字，自动列出对应的文件：

控制台出现的速度很快，资源占用大大小于 Launchy（听说 Launchy 新版本资源占用越来越厉害了）。12个皮肤虽然不算多，但都还比较好看。

Dash 和 Launchy 接近，都可以自动索引“开始菜单”等常用快捷方式集散地，而且它还原生支持索引 Firefox 书签、音乐目录。但因为是试用版，不能自己增加要建立索引的目录，也不能编辑修改索引选项。
一些特殊命令试用版中也无效：

增加关键词的方式很奇特：用 Add 命令启动下面这个界面，Open this 下拉菜单中是当前已经打开的程序清单，也可以手动选择其他文件和目录，然后设定对应的关键词别名和描述。但我不知道如何像 TAR 和 Executor 一样在资源管理器界面中为当前文件快速添加关键词别名。
试用版只能添加5个自定义关键词，但我没试能不能通过修改 Keywords 下面的文件突破限制。

Dash 有个特色，就是如果特殊命令可以带参数，比如 Play Music 命令，那么输入关键词之后再按 Tab 或空格，就进入参数输入模式，此时再输入你要播放的音乐文件名，支持模糊匹配：

另一特色是输入 /，就可以用方向键在控制台浏览本地目录和文件。截图无法很好的展示这个功能，自己试一试就能感受到这个功能很有想法。

最后，为何我要介绍一款收费软件？因为 Dash 的开发者之一声称要放出一个 new UNLIMITED TRIAL version：

I am one of the Dash Command developers. We are going to be releasing a new UNLIMITED TRIAL version to replace our old demo (which lags behind the current registered version by quite a lot).
这实在是让人期待。
但谁能告诉我 UNLIMITED TRIAL 是什么意思？不限时、不限次数还是不限功能？

补充说明：试用版的说明文档中说：

This version of Dash must only be run for the duration of the trial period.
FAQ 中说：

What are the restrictions in the trial version?

The restrictions at this point are a smaller set of commands, the user can only create 5 keywords, keywords and scanners cannot be edited. The trial is also time limited.

但我不知到这个 time limit 到底是多少，难道要我自己试？

开发者去年曾经说：

Unrestricted Trial: The next update we make to the trial version is going to be non-restricted, so that everyone can test all the features in Dash. We don’t have a set date yet, but I suspect it will be coming out around the end of Sept. or beginning of Oct.
结果到今年还没有推出全功能试用版…

床头捉刀
Table of contents

今天忽然从林妙可+杨沛宜为了国家利益的2合1想到《世说新语》中的一个小故事：

魏武将见匈奴使，自以形陋，不足雄远国，使崔季珪代，帝自捉刀立床头。既毕，令间谍问曰：「魏王何如？」匈奴使答曰：「魏王雅望非常；然床头捉刀人，此乃英雄也。」魏武闻之，追杀此使。
不过现在咱们到底是进步了，还不敢去追杀洋番蛮夷灭口的。

土摩托问：“你怎么知道刘翔的话是真的？”

只有在这个时候，刘翔的教练孙海平才出来解释。可是，没有独立记者的采访，没有其它知情人士的作证，没有公正的媒体环境，我凭什么相信你的解释？

奇怪的是，以前对我党所有的话都不相信的那些人，此刻突然又都热泪盈眶地信了。

很简单呀，这次即使是和最可信的阴谋论相比，孙海平们的解释也是更加靠谱的，怀疑不应该是专门针对特定的人或者团体。而且，目前谁的解释都不够透明，自然先选择更符合常识的那个解释了。

前几天逛商场，看到法藤的专柜上拿体育总局训练局合作伙伴的身份做宣传，估计是想让人以为这个产品多么多么有效，所以被体育总局训练局选中了。到它的中文站点（国内代理？）上就可以看到，其实最大的原因不过是有赞助罢了。
当然，训练局也说了，“希望经过特殊加工后的含钛法藤环类产品能有效缓解运动员颈部和手腕部的肌肉紧张，帮助运动员很好的进行赛前保护和赛后恢复。”但，“法藤作为一家以健康、科学为宗旨的企业，采用高新科技以及特殊加工的法藤产品”，真的是科学的吗？法藤宣传的所谓产品原理和效果，训练局真的进行科学验证了吗？还是说，打着科学的幌子，用赞助费开路，训练局就可以睁一只眼闭一只眼了？

对了，当时讨论法藤/法力藤/Phiten 产品的有效性时，刘翔还戴着呢。不知道这次受伤时还有没有戴法藤。

还能更扭曲一点吗？

一、扭曲
“我觉得很好看，如果你觉得不好，你一定是自卑到心理扭曲眼里只有缺点唯恐天下不乱抱外国爸爸大腿的汉奸小丑网特装逼犯”
“我觉得不好看，如果你觉得很好，你肯定是斯德哥尔摩症发作被政府洗脑沉醉于法西斯美学的阿Q愚民五毛”

外国媒体交口称赞
“连洋人都说好，我们就更没道理批评了”
“习惯自由散漫的洋人没见过这阵势才会被震住”

外国媒体批评、嘲讽
“这分明是眼红、嫉妒、别有用心、故意诋毁。”
“你看，果然在洋人面前把面子都丢光了吧！”

我就[马赛克]了！都能够不那么扭曲一点么？
表达你喜欢什么或者不喜欢什么，有必要对持不同意见者的身份和用意下判断么？有必要先看外国人的态度再决定怎么说话么？

二、风凉

无论何时都只会靠那些怪腔怪调的风凉话、妙语连珠的挑剔讽刺、以及那些永远抖不完的廉价的小机灵儿来表现自己高超才华和独到见解
对一切事物都冷眼旁观挑三拣四，做一个专业说风凉话的，永远翻着白眼儿打量世界上的每一件事物，觉得任何东西都入不了法眼
一辈子都不曾有那么一两次被什么东西打动，一辈子都只会挑剔而不会欣赏
不能老打算就靠这玩意儿扬名立万儿。别老拿别人当傻逼，别老觉得什么都瞧不上就高明了，更别生怕自己不来骂两句就显得自己没观点了
——东东枪《我由衷地觉得北京奥运开幕式很牛逼》

这样的文字很眼熟，连常去的几个宅男论坛互相喷起来，也会来上这么一手。
然而我的疑问是，那些不同的观点说出来个个都能够保证心平气和、温柔婉约、彬彬有礼么？器量小的，看什么都是尖酸讽刺、恶意捣乱，器量大的，就算真是面对不阴不阳的嘲讽也能平心静气，那么，谁能够判断的准哪些是挑剔找茬哪些是应该认真对待的批评呢？真都要行文温柔敦厚“有建设性”了，被“刺”者还能够听得进去？这我也要怀疑了。
我们都不是圣人，无论表达观点还是听取观点都可能受情绪等因素的影响，说话可能冲一点、阴一点，遇到不合意的声音难免觉得嘈杂刺耳，但即使一贯说话难听面目再可憎的人，他的言论也或有可取之处。风凉与否，我看并不是重点。
再者，不知道这里所指的人都是哪些，也不知道这“无论何时都只会”、“永远抖不完”、“对一切事物都”、“专业说风凉话”、“永远翻着白眼儿”、“任何东西都入不了”、“一辈子都不曾有那么一两次”、“一辈子都只会挑剔”、“什么都瞧不上”等等是如何观察或联想出来的，这真的不是因为自己喜欢或者赞同的事物被挑剔讽刺、被翻了白眼、被说了风凉话、被挑三拣四、被瞧不上，才愤愤不平的下此结论？唯一原因真的只可能是这些人看什么都不顺眼，而不是自己赞美的东西可能真的有那么糟糕吗？
至于“靠这玩意儿扬名立万”？我倒是想起了某类电视剧，剧中富有的家长们挥舞着钞票对主角们说，“你缠着我的儿子/女儿不放，不就是想要钱吗？…”诸如此类，我不明白认为自己能一眼看透别人动机的那份自信是从何而来的，这真的不是某种顺手使出来的先消解对方立场正当性的手段吗？扭曲到从对方的身上找动机不纯的理由这招谁都会，只是我没想到许多人只要觉得好用也就大大方方拿来用了。

三、评价开幕式
看开幕式直播的时候，感想已经同步发在 Twitter 上了，这里就做个总结好了：
整体来说比预期的要好，可能是因为本来就没有太高的期望吧。
华丽有余，气势很足，创意尚可，节奏偏慢，魂魄散乱。

最大的三个问题是：
1、脑残的导播和摄影——按照上面的逻辑，估计是反华人士打入党国的卧底，意图制造人民内部矛盾；

2、不可救药的解说——专业喧宾夺主人士？“我不露这个面儿照样把他的戏给弄筐瓢！

3、平淡无奇的主题歌——我还以为至少可以超过《北京欢迎你》呢，结果在解说的夹击下草草收场，一点都兴奋不起来。
