如果你看到的这一篇没什么问题，就说明升级到 WordPress 基本完成了！真是少见的顺利呀，不愧是以方便安装和升级、插件丰富的 WordPress。

准备升级 WordPress 2.0
Table of contents
    • AutoSave](http://twilightuniverse.com/2005/04/twilight-autosave/)、[Ultimate

因为安装 Ultimate Tag Warrior 的时候受了打击，所以下定决心把“滞销书”一次升级到 WordPress 2.0 版本，如果这几天 blog 无法访问，说明正在升级，或者升级失败，或者恢复失败… Orz
另外 RSS Feed 也可能出现异常状况。

先在 blog 上记录一下步骤，到时候也好知道到底是怎么死的。

1、备份数据库。
使用 Database Backup 插件导出数据库表。没有用 PhpMyAdmin，因为我是个懒人。到时候出了问题再搬救兵好了。按照常理，像我这样经常很倒霉的人出问题的概率是相当大的，何况还要偷懒…

2、备份文件。
虽然一些 blogger 说 wp-content 目录中的内容可以不备份，但我修改过默认模板的文件，所以要备份一下，升级后再看具体情况作修改。

3、停用所有的插件。
目前使用的插件有 Adhesive、Another Random Quote、FancyTooltips、SearchNow、[Twilight

AutoSave](http://twilightuniverse.com/2005/04/twilight-autosave/)、[Ultimate

Warrior](http://www.neato.co.nz/ultimate-tag-warrior/)、[wp-Define](http://www.zerohex.org/)、[WordPress Database Backup](http://www.skippy.net/blog/plugins/)、[Better Feed](http://frenchfragfactory.net/ozh/my-projects/wordpress-plugin-better-feed-rss/)。

4、下载 WordPress 2.0 文件。
下载地址。

5、删除文件。
保留的目录、文件：wp-config.php、wp-content 目录、wp-images 目录，必要的话，保留 wp-includes/languages/ 目录、.htaccess 文件，以及其他一些自己修改过的文件。
删除剩下的 WordPress 1.5.x 的文件。

6、上传 WordPress 2.0 文件。
最后上传 wp-content 目录，覆盖的时候要小心谨慎。

7、执行 wp-admin/upgrade.php。
然后就看人品是否过硬了。

8、在 Options 面板中重新设置 Permalinks

9、更新、激活插件。
更新 Ultimate Tag Warrior、Better Feed、SearchNow，试用 Random Quote、ThemeSwitcher。

“理想的书店”小问答
Table of contents

大头绿豆在豆瓣上问深圳的豆友“深圳有你理想中的书店吗？如果有，是哪儿？如果没有，那么你理想中的书店应该是怎样的？应该有哪些东西或者服务？”

0、你多久没逛书店了？如果很久，是因为什么？是和我一样，因为觉得深圳没有好的书店吗？

1、如果你常常逛，那末通常是因为什么？为什么不在网上买而要到书店去？

2、如果你在罗湖，有什么可以吸引你去一家南山的书店？坦白说，离得多远的书店是你能接受的？

3、你希望这家书店能有地方让你坐下来读书吗？顺便来点喝的？

4、如果它有一个足够的空间，你愿意和你的朋友们来这里聊天，或者只是听它的音乐吗？

5、如果你选择到这家书店发一个下午的呆，你能够接受的消费是多少？

6、你愿意把自己的一些旧书拿到这个书店卖吗，哪怕必须接受一些附加条件？

7、你愿意在这里买一些旧书吗，如果它确实讨你喜欢？

8、你希望在这家书店里看见一只猫或者什么别的动物吗？

9、你希望它是白色的吗？

在长沙的我也凑个热闹回答一下：
0、不算太久，不过比以前的频率低了。原因是工作的地方附近没有好书店，周末又一直很忙，又有不少书等着看完。

1、经常逛的原因当然是来瘾了，还有就是经常有让人感兴趣的新书到货。如果是问书店吸引我的地方，除了质量和价格，那就是书籍的口味、稀有书籍的数量、店主的服务态度、检索查询是否方便、阅读环境如何。不喜欢上网买书的原因嘛，我一定要在手里面捧着才算放心，看中了又不买太折磨人，况且买书也讲究氛围，而且说不定可以遇到有同好的美女 ;P

2、现在常去的书店离家里有好几公里吧？关键是要有直达的公交车，或者骑单车、寄存单车方便也行。

3、当然要有坐的地方，喝的倒不必，怕倒在书上不好交差－－可是好像经常一边吃饭一边看书哦，那好吧，咖啡、茶、果汁、牛奶都可以，但不要卖碳酸饮料。

4、聊天听音乐没兴趣，那已经是另一种氛围了。要是谁在我看到一本特别想翻翻看的书时硬拉着我聊天，我会发飙哦。

5、能不能够不消费啊？我的要求不算高，收费的服务对我来说是多余。

6、旧书？我会卖？那一定不是我的书…

7、当然要买。本地买不到，到外地去买都成。

8、无所谓，只要它不吃书不撕书就好，也不要太吵闹，否则必会招来人道毁灭－－我可不是无条件的动物保护主义者。有条件的话…长颈鹿怎么样？

9、是纯棉的吗？没有小熊或南瓜图案吧？(-_-#)
