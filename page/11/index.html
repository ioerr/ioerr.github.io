老婆常埋怨我说，服饰、鞋子等商品平常有打折优惠的时候，我总是说不需要、不急用，现有的够用，等到需要的时候再买，往往已经过了优惠时段，花费更多。

除了线下商品的打折优惠很可疑（没法跟踪历史价格）之外，我觉得还可以有个比较合理的解释：
急需时才购物，提升了需求的重要性和紧急性，一旦满足，那种久旱逢甘霖式的快感更甚一筹。

说得好像很有道理呢，连我自己都快要相信了。

必应输入法自定义无零声母双拼方案

微软的必应输入法支持双拼的能力一直不怎么样。
最开始是根本就没有双拼功能；
后来增加了双拼功能，却只支持微软双拼方案；
再后来多了几个方案，却既没有应用最广泛的自然码方案，也没有近年来比较流行的小鹤方案；
现在终于可以自定义双拼方案了，却还死板地要求必须为零声母分配一个按键，而且多字声母必须以零声母开头……

从升级日志来看，快要一年没有维护更新了，估计官方也不会有什么大的升级，还是自己动手吧。

首先可以新建、另存一个双拼方案；
然后去用户目录下找到 Roaming\Kunlun\double-pinyin 目录；
其中应该会有个后缀为 imedps 的 XML 文件，用文本编辑器直接编辑；
文件改为以下内容：

  </br>

<double-pinyin name="小鹤双拼" id="ce378851-922f-401c-966a-1a2ede2f57d6" xmlns="http://schemas.microsoft.com/ime/double-pinyin/1.0">  </br>
    <consonants>  </br>
        <map syllable="ch" key="i"/>  </br>
        <map syllable="sh" key="u"/>  </br>
        <map syllable="zh" key="v"/>  </br>
    </consonants>  </br>
    <vowels>  </br>
        <map syllable="a" key="a"/>  </br>
        <map syllable="ai" key="d"/>  </br>
        <map syllable="an" key="j"/>  </br>
        <map syllable="ang" key="h"/>  </br>
        <map syllable="ao" key="c"/>  </br>
        <map syllable="e" key="e"/>  </br>
        <map syllable="ei" key="w"/>  </br>
        <map syllable="en" key="f"/>  </br>
        <map syllable="eng" key="g"/>  </br>
        <map syllable="er" key="r"/>  </br>
        <map syllable="i" key="i"/>  </br>
        <map syllable="ia" key="x"/>  </br>
        <map syllable="ian" key="m"/>  </br>
        <map syllable="iang" key="l"/>  </br>
        <map syllable="iao" key="n"/>  </br>
        <map syllable="ie" key="p"/>  </br>
        <map syllable="in" key="b"/>  </br>
        <map syllable="ing" key="k"/>  </br>
        <map syllable="iong" key="s"/>  </br>
        <map syllable="iu" key="q"/>  </br>
        <map syllable="o" key="o"/>  </br>
        <map syllable="ong" key="s"/>  </br>
        <map syllable="ou" key="z"/>  </br>
        <map syllable="u" key="u"/>  </br>
        <map syllable="ua" key="x"/>  </br>
        <map syllable="uai" key="k"/>  </br>
        <map syllable="uan" key="r"/>  </br>
        <map syllable="uang" key="l"/>  </br>
        <map syllable="ue" key="t"/>  </br>
        <map syllable="ui" key="v"/>  </br>
        <map syllable="un" key="y"/>  </br>
        <map syllable="uo" key="o"/>  </br>
        <map syllable="v" key="v"/>  </br>
    </vowels>  </br>
    <solo-vowels>  </br>
        <map syllable="a" key="aa"/>  </br>
        <map syllable="ai" key="ai"/>  </br>
        <map syllable="an" key="an"/>  </br>
        <map syllable="ang" key="oh"/>  </br>
        <map syllable="ao" key="ao"/>  </br>
        <map syllable="e" key="ee"/>  </br>
        <map syllable="ei" key="ei"/>  </br>
        <map syllable="en" key="en"/>  </br>
        <map syllable="eng" key="og"/>  </br>
        <map syllable="er" key="er"/>  </br>
        <map syllable="o" key="oo"/>  </br>
        <map syllable="ou" key="ou"/>  </br>
    </solo-vowels>  </br>
</double-pinyin>  </br>

然后保存，去必应输入法双拼方案设置界面选择“小鹤双拼”即可。

运气
Table of contents

曾经觉得自己的运气特别不好，倒不是怨天尤人或者为失败找借口，而是总可以遇到许多匪夷所思、非常巧合的事情，使得本来一帆风顺的事情横生枝节。
不但如此，发生的必然性几乎都是可以预测的，只有形式无法预料，以至于有好几次我都可以预言将有意外发生，而果真马上应验，让朋友大吃一惊，以为我是半仙。

然而，后来我才明白，所谓的运气不好，只是因为我总是注意到运气不好的事情而已。
其他人同样也会遇到各种各样的倒霉事情，区别只在于，有的人只会抱怨老天爷不公平，有的人会觉得概率分布很神奇，而有的人则把种种不顺视为理所当然。

把做一件事的过程画下来，就像波浪起伏的曲线。
还在曲线半路上的人难免心有彷徨，左右看看已经顺利走到曲线终点的他人，只看得到最终的结果，容易将挫折和困难视为运气不佳。
而其他人一路上的艰辛并不更少，只是那些倒霉的事情阻止不了其奔向终点的决心。
套用一句特别鸡汤的话，内心强大的人，运气往往也不会差。
即使个体的运气有差别，以一般人的经历和环境而言，也难以成为决定命运的最重要因素。
