世界佛教论坛开幕式
Table of contents

上午看第二届世界佛教论坛开幕式，开幕演讲中“和谐”这个词出现的频率实在是太高了，每听到一次就起一次鸡皮疙瘩，相当难受。
其实高僧大德们谈和谐是一件很正常的事情，要是放在10年前我不会有这么大的反应。
为什么现在视之如毒虫猛兽避之犹恐不及呢？
大概是因为现在这个词已经被污染了，变得空洞、模糊、虚妄，什么场合都可以用上，又什么实际意义都没有——如果不是形容词作谓语或使动用法的话。
这让我想起了哈维尔论述后极权主义社会的时候所举的那个例子，橱窗里“全世界无产者，联合起来 ! ”的标语，替换成“要建设社会主义和谐社会！”，有本质的区别吗？

如果现在包围我们的这种很少有人真的相信、更少有人真的努力去实现、已经背离了原义的“和谐”，最终慢慢取代了中国佛教徒心中的和谐概念的话，鸡皮疙瘩就远不足以表达我的悲愤了。

十一世班禅额尔德尼上台发言的时候，台上诸人中数他的年龄最小，他的动作举止都十分有礼貌有分寸，英语发言讲得也很好（至少比前面的泰式英语强）。

很奇怪的是，电视转播时好像是配的同声翻译。其实发言的几位高僧都是照着稿子念，完全可以事先拿到发言稿翻译好吧，毕竟本来同声翻译要求就很高，还有这么多佛教专有词汇和简短而不简单的中文。

十一世班禅发言之后，中央电视台的转播马上就切到了西藏农奴解放50周年纪念节目，让人感觉不是滋味，总觉得央视转播世界佛教论坛开幕式就是为了给西藏的宣传开路，用完了就丢到一旁了。

最后，世界佛教论坛的网站晚上已经进不去了。虽然上面看不到多少即时新闻，还不如去门户，但也挺讽刺的。

CECT S610
Table of contents

新手机到手，选择了性价比比较高，体积也没有一般的 PPC 那么吓人的 CECT S610（对应的 GSM 机型是 i-Mate 的 PDAL）。基本不需要玩什么游戏，也不指望拿来看电影、听音乐、拍照片，所以对硬件的要求不高；折腾得不多，所以不刷机也能用。
到手之后，除了传说中的电池续航能力是最大的问题之外，其他一切倒还好。

到手之后发现自己对 Windows Mobile 很不习惯，而且常用的应用不可或缺，于是立刻安装趁手的工具：

Total Commander for Pocket PC
据说 Resco Explorer 很好，但是对 TC 有感情，又是免费的版本，当然用 TC。

点讯输入法
自带的输入法始终不够用，而同时支持26键布局和双拼输入的点讯就成了首选，如果 A4有双拼方案才会再考虑。
点讯划词输入的创意不错，双拼方案也可以自己定义。

Haali Reader
电子书阅读软件选择太多了，好用、易用、免费、轻量级的随便挑一个就行。

UCWeb
Opera Mini 用久了换换口味，首选速度快、支持多窗口、免费的 UCWeb。

无限闹钟
自带的闹钟功能太弱，用功能强大、比较稳定的无限闹钟代替。好像作者已经停止了开发，但目前的版本已经足够用了，甚至多了许多不需要的功能。

快速菜单
既然没有手机上的 Executor，那就好好整理一下 Windows Mobile 的开始菜单，不要迷失在 \$tart 的汪洋大海中。

PockeTwit
Twitter 和 FriendFeed 上多位朋友推荐的 Twitter 手机客户端，感觉和网页版的要结合起来用才行。

ListPro
本质上只是一个清单管理工具，但以它为基础却可以实现丰富多彩的应用。
收费软件，分桌面和手机客户端，30天试用。

Jbed Java 模拟器
需要使用 Java 程序的时候才用得到，目前只装了个 Gmail for Mobile。

蒸菜
Table of contents

最近加班多，于是经常在公司附近的浏阳蒸菜馆吃饭，感觉性价比很高。

最大的优点是上菜速度快。
反正都是已经做好的菜，一直用蒸的方式保温，一点完菜就可以开始吃，很适合时间紧张的人士。

然后是价格还算便宜。
荤菜6元，素菜2元，最低消费每人6元。一个人吃也就是8元左右，几个人一起吃更加划得来。

配套服务齐全。
米饭无限量供应，附茶水、卷筒纸、牙签，有时候老板还附赠家乡带来的蒿子粑粑。

食谱还算健康。
至少每餐可以吃到素菜，不喜欢蒸的素菜还可以要店家临时炒菜，比如菠菜和小白菜。
荤菜有红烧猪脚、红烧排骨、蒜苗鸡丁、鸡杂、梅菜扣肉、腊肠、玫瑰排骨、红烧肉等等，素菜有芹菜香干、白萝卜、茄子、豆角等等，蒸蛋算素菜。相比之下，经常吃面、粉、炒饭或者煲仔饭，食谱就比较单一了，可以选择的余地和组合也不多。

总体来说，除了卫生状况和环境之外，蒸菜比洋快餐更适合我，吃肯德基的时候，光是汉堡×2+美禄就差不多25元了，到头来还吃不饱，素食偏少。

我觉得蒸菜这种形式其实应该很适合中式快餐，至少可以解决上菜慢的瓶颈，比如现在就有连锁的“华人蒸菜馆”在做中式蒸菜快餐的生意——然则我有过一次在华人蒸菜馆空闲时段等了40分钟才吃到饭菜的经历…
