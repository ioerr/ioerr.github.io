英文打字辅助工具
Table of contents

时不时会打一些英文，有的是非常用单词和专有名词，容易出错，所以想偷懒找个英文打字辅助工具，类似编程时自动提示函数名称等等的。

印象中有金山英文打字助手，但以前用的时候感觉不好，具体什么原因也忘了。

Google 了一下，找到两个国外的软件是 Smart Type Assistant 和 As-U-Type，可惜都是收费的，而且各有缺点。

国内的英文打字助手做得也不错，仍然是收费的。如果不是在 Firefox 中无法正常使用让我觉得68元有点太贵，我还真有购买的想法。双击 Caps Lock 激活的设置很聪明。

免费的午餐也不是没有，Typing Assistant 就很不错，选项虽然不多，但需要人工干预的地方很少，除了没有现成专业词库外，比前面两个收费软件反倒更为顺手。

可惜还是有致命的问题：
1、在其他窗口打开某些中文输入法时在当前窗口也会失效，我不可能在使用前还挨个检查应用程序窗口，把输入法关掉；
2、还是不支持 Firefox；
3、不能够自定义生效的应用程序，例如我在 TypeAndRun 时就不希望它跳出来。

不知道 AutoTyping 和 Typing Assistant 是什么关系，感觉提示界面非常接近，不过功能还是强大一些。
分为免费版和收费版，免费版只对写字板和部分 IM 有效，故名 Auto Typing IM，收费版其实也好不到哪里去，因为列出的可支持程序实在是很有限。
差点忘了说，AutoTyping 在我这里工作不正常，每次只能够根据输入的最后一个字母给出提示，也就是说，完全无用…
