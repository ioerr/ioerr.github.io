《中国愤青们的真实生活》评论摘选
Table of contents

新浪登了一篇《瞭望东方周刊》的文章，《中国愤青们的真实生活》，被不少人指责“不够客观”、“别有用心”、“先入为主”、“把‘愤青’等同于‘反日’”，不过在这里不说这些，我只想选出一些有意思的评论，其中有我支持的也有反对的：

允许自己观点存在的同时，也应该允许别人不同的观点存在！\ －－不折不扣的汉奸文章！ “一个小国能侵略那么多的国家”，这里所包含的大道理，不是FQ所能明白的，因为一旦明白了其中的道理，其也就不再是FQ了。…沉迷于大国梦，恰恰是FQ的一个标志特征。 中国这样一个大国需要愤怒的青年存在！ 多看些书吧，你会明白中国不是由于“愤青”太少而受人欺侮，而是实实在在明白道理研究问题的人太少才落后挨打 中国的愤情不是多了而是少了，在这共问题上装冷静、说忧思，要不是闹噱头，要么是什么人的枪手。 《瞭望东方周刊》以前还有点人性，现在怎么变成禽兽了！ 正是与日本人的接触,所以更了解日本人伪善的一面.也了解抵制日本品牌的必要(是日本品牌,而非日货,以免被那些"理智"的人们说有些国货也有日本配件).也更鄙视大多数日企里的所谓国人!!! 我看那些‘青粪”都是真正的爱国者，都是那些人不喜欢他们？我想除了美国和日本人或汉奸之外，真正的中国人都是喜欢他们的，中国的希望就要靠他们的愤怒和呐喊来帮住政府来实现强国战略我喜欢中国的愤青们。 愤青的存在，可以阻止并揭露美国等企图破坏中国稳定的阴谋，使大多数中国人不上当，还让他们看到中国人不是好欺负的，令其不敢轻举妄动 我们曾经提倡以史为鉴,面向未来,给过日本改恶从善的机会,它不珍惜,现在反而在抹杀历史,和中国乃至世界人民为敌,对于这样的国家和民族,我们为什么不能愤青! 就问你一句话,你是不是中国人呢? 愤青者，愤怒的青年也！为何愤怒？爱国主张无处伸张，不爱国现象处处显现！凡爱国者，无不切身体会和关注国家的内忧外患；无不对损害国家利益的行为深恶痛绝；无不感到报国无门；无不痛恨治国无方！他们是上午十一点钟的太阳。他们的言行深刻地影响了普通民众，同时也极大地撼动了有关利益方面。他们缺乏的不是辨别是非的能力，也决不缺乏与官僚资本主义斗争的勇气！！ 愤怒的不只是青年，愤怒的是全中国勤劳勇敢的人民！当然，汉奸一向是不纳入人民之列的。 文中的提及的“右派”大多信奉自由主义，即个人的信仰，思维，言论的自由。那么好，作者有什么理由去干涉愤青的思维。即便有些偏激的愤青，这是他的自由，只要是合法的，有什么不可以。见过太多的号称右派的人，做着违背右派教义的事。 如果《瞭望东方周刊》不对爱国青年道歉，呼吁对其进行抵制 所谓的愤青都是一群大傻冒，10年后他们自己都会脸红和后悔。\ 十年后他们有人也许战死，但决不会脸红或后悔。 作者应该被推上道德法庭 作为民众中的一分子,我情愿选择秦皇汉武,而不会选择汉献帝晋惠帝之流! 愤青是社会的精英，是有识的先进人士，是社会强硬的保障，谁反对愤青就和反对梁启超和百日维新东林党人一样的愚昧！ 一个伟人说过：学生运动天然合理。学生纯真，没有私利，不象文章作者有那么多的“理性”。他们“理性的”为了吃饭当汉奸，为了实现自己的价值当汉奸，为了曲线救国当汉奸，为了反对当权者的错误当汉奸，为了-----许许多多的理由而当汉奸。这就是中国多汉奸的原因。\ 学生运动天然合理。为那些尚未成熟的学生们的爱国激情欢呼吧，他们是中国未来的脊梁。\ 鄙视那些理性的“精英”们。他们在和平时期会夸夸其谈，国难时期是不能指望他们的。 最后我想说的是，如果国家需要，如果哪天和日本有一战，最积极投军报国的人一定是这些愤青。 -------------------------------------------------------------------------------------- 其实国家现在也非常需要志愿者报名参军守卫祖国辽阔的西北边疆,你......? 愤青是民主的自由的，人权的，反愤青，就是反民主 民意第一，没人去和你谈什么理论，也没有那工夫，大家气愤的事情，就是民意所向！ 也许学识不如你，也许我们“不学无术”，但当战争打响的时候，英雄是我们！汉奸就是你！ 因为中国有你,所以日本人总想征服中国,因为你是他们的良民!在对日作战前,清洗一批汉奸我双手同意! 中国需要愤青，需要振奋民族精神！过激、漫骂言行只是刺激那些麻木不仁者的手段。 看看这些评论吧！反对愤青的言论充斥着谩骂和诋毁！支持的言论却理性和和缓得多！公理自在人心！！ 楼下这么多人叫嚣着战争，你们知道什么是战争吗？你们又知道每次战争会带来多少痛苦吗？ 爱国是没有道德界限的

找书（PHP/HCI）
Table of contents

今天跑了一趟书店，想找几本 PHP 和 HCI 方面的书。PHP

书籍是入门级别的即可，但不要那种抓几个高校学生东拼西凑弄出来的劣等书；已经有较老的影印版

教材，所以希望找到一些跟得上时代、深入浅出的中文书，最好是既有详尽的实例说明，又有深厚的理论基础。
结果颇让人失望，无论大书店小书摊这两类的书都挺少，HCI 的只在一家计算机技术专业书店中找到两本还算有分量的中文书，粗粗翻看了一下，质量尚可，但和我的要求有差距，还不足以让我掏钱；PHP 方面的书按道理不至于这么少，但我仅在不少书店的角落里，一大堆 ASP、JSP、XML/XHTML、.Net 等等书籍的附近找到那么几本，还是质量不过关，或者是“实战入门”这种抄一些介绍和代码的骗钱货。
或许真的该上网订购，可是我从来都是不自己动手翻翻书就不放心出血的人，就算可以听取别人的评价，也代替不了我自己的判断。
在豆瓣看了一下，有 Rasmus Lerdorf 参与编写的《PHP 程序设计/Programming PHP》可以试试，不成的话，就去 PHP.net 看手册算了。至于 HCI 的，依然没有什么收获…

许久没有登陆 Orkut，都快忘了 Google 还有这么一个服务。今天收到了 Tinyfool 的邀请才重新上去看了看，结果时不时就来个

Bad, bad server. No donut for you

Unfortunately, the orkut.com server has acted out in an unexpected way. Hopefully, it will return to its helpful self if you try again in a few minutes.

It’s likely that the server will behave this way on occasion during the coming months. We apologize for the inconvenience and for our server’s lack of consideration for others.

究竟还有没有生命力啊？

对我这样时时开着 Gmail 的懒人来说，Orkut 远没有 Google Groups 来得方便，也不必通过它去交友什么的。倒不如把 Groups 的 Web 方面再增强一点…

就连成员达到98万7500的巴西社区，一天的主题数也不到10条，加上回复也不过20多条而已。话说回来，怎么巴西人对这个服务的热情这么高？好几个巴西的社区，其成员人数都名列前茅，难道有诈？可这也没什么好处嘛。
