推荐 WordPress 主题
Table of contents

用惯了 WordPress 的默认模板 Kubrick，也会有更换模板的念头。不过一来我很懒，二来看得到页面的不多，三来合我心意的主题实在不多，所以看是一切照旧。也不排除哪一天心血来潮唰的一下就更换了主题，或者弄个主题切换插件，所以先列出几个我觉得挺优秀的候选者，也方便其他 WordPress 使用者作出选择。

◆大头的 ChinaRed。
配色和图案都非常棒，缺点则是默认的中文字体太小。相比之下，同一站点切换使用的 OrangeSky 就差劲许多。

◆Quentin 1.0
让人联想到牛奶加巧克力,但我喜欢巧克力的味道胜过它的颜色…

◆Contempt Wordpress Theme
看上去就是 Kubrick 的近亲，不过更富有现代感。

◆ADVENTURE theme
颜色不花哨，采用我喜欢的 Georgia 字体，看上去有点正式出版物的样子。但#600的颜色真的不好看…

◆Fahlstad.se 的一系列主题
就一个词，养眼。所以要从中间挑一个特别出众的也不是容易的事情。

◆Ocean
新鲜出炉的一个主题。

TinyPic
Table of contents

TinyPic 是一个提供临时上传图片服务的网站，以前我曾经用它上传照片，因为操作简单、速度和稳定性也不错，图片生存周期也往往比网站承诺的要长，因此使用的人越来越多，无聊的时候还能够随机查看其他人上传的图片，往往有意外的惊喜和乐趣。
