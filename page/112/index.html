语音和视频聊天？
Table of contents

最近我妈在老同学教唆下准备开始用 Skype，于是今天去买摄像头和麦克风。可以使用 VIP 卡的商城里面都没什么价廉物美的好货，电脑城春节期间全部休假，刚准备去国美之类的家电城，就接到个十万火急的电话赶回家为忘带钥匙的家人开门，于是暂时作罢。

不论是之前的 QQ、Google Talk 还是现在的 Skype，我都很不喜欢使用语音和视频聊天的功能。除了工作时打电话太多导致下班后对语音聊天完全没兴趣之外，不方便多进程同时进行、快速切换和无法保存完整的记录也是主要原因。

最重要的原因则可能是，我不喜欢语音和视频聊天时可能出现的因为要做其他事情而不礼貌地冷落对方、长时间沉寂的尴尬和被路人甲听到谈话内容诸如此类的情形。
作为纯文本党人，到底还是文字的方式最合适啊。

交响情人梦
Table of contents
    • 说起来最近几个月来有关注过的高雅星（《怪物》）、坂野真弥（《无厘头森林之第一次接触》）和上野树里都是短发类型的呢…嗯，关于前面两个，我真的不是萝莉控

花了一个晚上下载，在2天内的空闲时间看完了《交响情人梦》。虽然之前也有听说，但是真正想看是因为看到了屁屁体操的片段再找了剧情介绍的原因。

本来很少看电视剧，无论美剧、日剧还是韩剧，最近几年也就看了《人人都爱雷蒙德》，然后是零星的 Monty Python 短剧、老友记、24之类的，除了 Monty Python 的电视剧集是弄不到片源 ，其他的都没有一口气看完或追下去的动力。这次一鼓作气看完《交响情人梦》还真是叫人意外。

首先要感谢只有11集的容量，不必像别人追美剧那样痛而不通或绵绵无绝期，短时间内也可以消化完毕。

然后是长相有点像胡歌但似乎更帅的男主角扮演者玉木宏，瞪眼、白眼、僵直和嘴角抽动效果不错，把主角的无奈演活了，比某些偶像剧的脂粉美型男也阳刚一些。

饰演女主角野田惠的上野树里是最大的亮点，虽然不属于第一眼美女的类型，但是就连被殴打的表情都很可爱啊（可惜后面几集随着这类击飞的镜头减少娱乐度也有所下降…呃，我有虐待狂潜质么？:023:），第5集出现的獴造型更是有杀伤力。至于肮脏、凌乱和好吃的毛病对电视剧角色来说都不算什么吧——现实中能够让我实在看不下去而主动想要帮忙动手打扫卫生的人至今还没有达到过2个呢。
从网上找到的图片来看，上野树里的长发造型体现不出她的灵气，还是刚刚睡醒时的上翘短发最适合她啊。

说起来最近几个月来有关注过的高雅星（《怪物》）、坂野真弥（《无厘头森林之第一次接触》）和上野树里都是短发类型的呢…嗯，关于前面两个，我真的不是萝莉控

最后一个理由是很久没有认认真真听古典音乐了，虽然电脑上可以播放，但是听的时候如果还在做其他事情，这种三心二意的态度听古典音乐是听不进去的，乐趣还不如口水歌。而看这部电视剧也算是稍稍重温一下莫扎特、贝多芬、勃拉姆斯、肖邦、舒伯特、门德尔松等人的音乐呢——况且演员们还算有底子，不至于像某些粗制滥造的电视剧中演员们不知所谓地锯木头和弹棉花。
