Zizou 回归法国队

Zidane 宣布重返法国国家队，和他一同回来的还有 Makelele。

听到这个消息，我的第一反应是很开心。没有了 Zizou 的法国队总像缺少了什么独特的气质一样，而他能够多打一场比赛，我们这些球迷也就能够多得到一次享受。
当年 Maradona、van Basten 们正风光的时候，国内的球迷大多数是无法看到现场直播的，除了世界杯、丰田杯这样的赛事，大部分的信息来源只能够是报纸和事后的录像回顾，而这一辈的球迷却能够坐在电视机前，在无法预知结果的情况下目睹这些顶极球星创造一段段传奇的惊心动魄的全过程，以及欣赏由体育竞技升华成为的赏心悦目的艺术，相比之下实在是幸福得太多。也因为如此，现在的我，是十分珍惜 Zizou 这样的球星在挂靴之前的最后表演的，等到哪一天，这块阿尔及利亚和马赛的，同时也是法国和世界的瑰宝，永远告别了他曾上演过让世界为之瞠目结舌、为之击节叫好、为之如醉如痴、为之心潮澎湃的绿茵场，我们这一代球迷便又只能够不厌其烦地向后来者讲述那仅存在记忆中的传奇了－－即使有数不胜数的比赛录像，他们又怎能理解…

然而另一方面，我又不希望 Zizou 重返国家队。
首先，Real Madrid 已经打算在新赛季将 Zizou 从防守的功课中解放出来，让他放手一搏，在这个时候回到国家队，等于是把本来就在一系列商业比赛中储备不够的体能提前耗尽，那么无论这个赛季 Real Madrid 是否有出色的表现，Zizou 都很有可能做不到始终维持一个较高的水准了，也许这样对国家队和俱乐部都不利－－尽管 Makelele 的被迫出走和 Giuly 的存在让我并不偏爱 Real Madrid。
其次，如今的法国队虽然步履维艰，连世界杯预选赛小组赛出线都有很大的危险，但就算 Zizou 回归，法国队便能够猛地复兴？2004年欧洲杯上 Zizou 在的时候就分明既没有解决好后防老化的痼疾，也没有理顺前场进攻的乱麻，还扶正 Silvestre 这个定时炸弹，已经说明 Zizou 并非万灵药，凭什么相信 Zizou 回归后法国队就能够一改颓势焕发生机？
捞足了荣誉的法国队如今正在进行低潮期的更新换代，Domenech 虽然没有什么好手段用活这批年轻球员，至少全面年轻化的道路不应该逆转，Lemerre、Santini 就是保守思路的前车之鉴，86年之后的那支法国队也是活的教训－－一时的成绩不佳并不可怕，因为急功近利而耽误新一批球员的迅速成熟才会对未来产生深远而恶劣的影响。也就是说，Domenech 先生的“青春泉作战”绝不该半途而废：如果他歇菜了，那么请换个教练继续带领年轻的蓝军匍匐前进，现在，原本信誓旦旦的 Domenech 先生眼看要玩不转了，就让球队昔日的灵魂再回来试试看，等于是否定了之前所有的努力，无论成败，都对法国国家队的年轻化有极大的负面作用。而且 Zizou 并非一般的只是在经验上有利于球队的老队员，他必然重新成为球队的大脑，而且会对球队的战术设置有决定性的影响，一旦他日后再次退出，国家队又会面临重新洗牌从头再来的尴尬局面，而这一进一出之间，又是一代年轻人的前途被扼杀了。
至于世界杯预选赛出局的危险，我这样的法迷倒并不怎么担心，90、94年两届世界杯法国队都倒在预选赛阶段，法迷并非承受不了，与世界杯两度不进相比，更令人痛心的是球队战术思想的混乱，看不到整合成功的希望和一大批优秀球员的才华在保守与首鼠两端中被挥霍一空…

当然，完美的结局也不是不可能，Zizou

Book Crossing
Table of contents

呃，比较火星的内容了，不过可能还有不少人不知道，所以多嘴一下：）

如果你捡到一本找不到主人的书，你会怎么处置呢？阅读，丢掉，还是读完再丢掉？有没有想过这本书也许是它的原主人故意留给别人并要下一个人继续传播呢？

Book Crossing 就把这样一个好玩的点子变成了行动，虽然听上去更像《读者文摘》里面老掉牙的小资故事：
找一本你读完了的书；
在 Book Crossing 注册，得到一个唯一的 BCID 号码，然后制作成标签贴在这本书上；
最后，把它交给其他人，比如你的朋友，比如偷偷放在公园、咖啡店这样的公共场所，或者做善事捐出去，然后等着下一位读者通过标签找到 Book Crossing 后，让大家知道这本书旅行到哪里了。如果你打算留给陌生人，最好是在注册后，做好 Release Notes，以便这个陌生人也能通过 Active Releases List 找到这本书

如果你觉得这件事情很有趣或者很有意义，不妨试一试，也许能够通过这个方式交到不少同样爱书的朋友：）当然，把心爱的书弄丢了，也只能够认命了。
你可以自己制作一款中文的标签上传到 Book Crossing，这样就不用担心其他人因为不懂外语而错过交流和再次传播的机会。

我看了一下传播次数最多的书籍，Douglas Adams 的《The Hitchhiker’s Guide to the Galaxy》以22次排在第79位。

顺便说一句，Book Crossing 在2005年的第9界 Webby Awards 中获得 People’s Voice Awards 中的 Best Community Website 与 Best Social Networking Website 奖项。虽然这个网站很早就出现了…

Racing Club de Paris #6

就在第二天，Dijon 俱乐部对 Sébastien Bertrand 的报价传到了我的手中：\$200K。
这对一个只有\$600左右资金的俱乐部来说不算少，而且，既然 Bertrand 打定主意要去一家大俱乐部，那么同样在 National、球场只能够容纳8000人的 Dijon 估计也不是他的终点站，那么不妨再从他身上压榨多一点价值吧－－抽取下次转会20％的转会金，但愿明天一觉醒来，Real Madrid 或者 Dortmund 烧钱把他买到手…哦，我又盼面包牛奶到发疯了…

另一份文件是关于27岁的阿尔及利亚籍自由球员 Omar Belbey 的，位置是防守型中场，得到的评价颇为一致：“是个天才，应该成为阵容中的长期成员，有机会出人头地”，最后附着的一张小小的小到我差一点就没看见的纸条是他的签约条件：周薪\$17K，签字费\$75K，助攻奖金\$130。
我觉得后脖子有一种火焰灼烧的奇怪感觉，如果这个时候那个提交报告的球探出现在我面前，我是绝对不会顾及自己的形象，用一记庐山升龙霸将其击打至阿尔及利亚的－－这球员的周薪要求与目前全队的周薪总额持平，他还介绍给我，是要我以身试探 Gilles Dumas 主席有没有学会传说中的瞬狱杀吗？(-_-#)

相比之下，Thierry Laborieux 推荐的冰岛人 Hallur Kristján Ásgeirsson 就比较符合我的要求，射手出身，Finishing、Stamina、Technique 都有一定水准，Aggressive、Agility、Work Rate 这些有利于抢点型射手的属性也相当不错，就是要求的周薪接近工资上限了。还是削减削减以后看看球员的反应吧。

除了寻找新球员，还得在临场指挥中合理利用好队内的现有资源。French National 的比赛只允许带3名替补，如果咬咬牙把替补门将从板凳上裁掉的话，也要尽量保证每一次换人都是卓有成效的，但再好的教练也不可能保证所有3名替补刚好就能弥补阵容的不足，距离完美的解决之道最近的方案是手头囤积几个万金油式的球员，无论做主力做替补都可以随时出现在任何位置，并补上最短的那一块木板，将整个球队从崩溃的边缘拯救出来。
理想虽好，总要向现实低头，手头的球员大多只能胜任单一的位置，如果锋线、中场、后卫各有一名替补，也无法兼顾中路与边路，因此恐怕时常会让人有捉襟见肘之感。例如第一场比赛中，Djeme 在 D R 的位置上无法正常发挥，我的替补中却只有 Izquierdo 这一个 D C 能够投入到后防线上，以至于不得不让同样身为 D C 的 Lalisse 临时改打 D R，而在左后卫 Soulas 也无法承受对方的攻击时，我的手头却只有 Poisson 一个前锋可用，要么冒险临时变阵，要么死撑。
所以，引援目标还不能够少了一个多面手…啊，Steven Reid～Christoffer Andersson～快来主动投效我吧～
