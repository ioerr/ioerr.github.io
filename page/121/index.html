旁观者的饶舌
Table of contents

之前 Zola 和 Hung 两个人“PK”，两边的动作像是耍太极拳，舞了半天也没有在关键的地方见血，所以有不了了之的迹象。

不料 Zola 的这一篇文章再次谈到网谈这个话题，把米随随给惹毛了，后者的表现近似于骂街，且有歧视的嫌疑，实在是让人遗憾。

我希望这场混战能够回到实际点的层次上面来，比如不妨深入讨论网谈究竟是不是“流氓”？
网谈是不是曾经和正在抄袭文章？是不是以商业 blog 的身份违反过其他 blog 创作共用协议中“非商业用途使用”的条目？这应该是一个能够取证且有确定答案的问题。

其他的争议问题，有的旁人无法取证，有的没有统一的评判标准，可以暂时搁置，免得分散精力还无法下结论。至于为人态度如何、80后独生子女普遍存在的问题等等，有的纯熟个人感情，不宜作为支持或反对的理由，有的属于视野狭窄又一厢情愿的老生常谈，得要从人的思维方式上下手，真要深入地说，上可至天文地理下可至鸡毛蒜皮，与主题无关了。

然后对米随随说几句。Hung 之前给 Zola 写过两篇文章，至少我觉得算是比较诚恳的。
他谈到“改革开放之后的独生子女环境所造就的一代人”缺乏责任感，这个结论从其前提、推论、证据、应用到延伸，我都是不认可的，但强调责任感总是有积极意义的。

责任感，不仅仅是你在现实中成就什么事业、影响多少人，在网上的言论同样体现一个人的责任感。你的那些发泄的文字写出来是很爽，还有不明事理的朋友起哄和叫好，但文章中折射出的粗鄙和歧视色彩，影响很坏。这样的言论，和 Hung 所说的无聊、肤浅、不成熟有什么本质区别？甚至更加恶劣。

从留言来看，你的这些文章也影响了 BXNA 的 RSS Feed 质量，让读者发出抱怨，这难道也是对群组负责任的表现吗？

你又说 Zola 总是在指责别人，他没有本事搞好自己的事情，没有权利指责别人。
但我不认同这样的逻辑。一个人有没有权利批评，批评得对不对，不是完全由他自身的情况决定的。如果说的是事实，难道也可以否认吗？如果说的确实有道理，难道也始终比不上那些“有能力到不用为生存发愁”的人说的胡话？
因人废言，可能让你只能够接受说好话的人，可能让你错过有价值的批评。

有则改之，无则加勉，我之前希望 Zola 以这个态度面对 Hung 的回应，现在也希望你能够以这样的态度面对 Zola 的批评。或者觉得价值观不同，拿出来平心静气的讨论也可以，一上来就说对方没权利没资格，还外加污言秽语，一来别人果然觉得你的那套价值观不怎么样，二来未免显得太小器吧，其三，这算不算 Hung 所批评的“缺乏与人交往中的正常心理”、“盲目偏激”呢？

最后，有人在 Zola 的文章中留言说，“其实有很多更有意义的事情等着我们去做呢”。这是一句完全正确的废话。除了救人性命和灵魂、探悉人生和宇宙的奥秘之外，恐怕也没有几件事情不符合这句话了。
但是，当下有意义的事情，哪怕再卑微，我们如果有能力还是应该去做，不是吗？勿以善小而不为啊。

如果这句话用来逃避批评，那么就和“其实还有很多罪行更为深重的人等着你们去抓，为啥只抓我”的辩白一样可笑了。人家作恶的时候可没有因为“还有更恶劣的事情等着我们去做，所以这件坏事就不做了”呢。

无限量的图片存储空间；

最多建立3个 photoset，相当于相片集；

如此看来，Flickr 提供的免费服务和同类服务相比没有特别让人心动的地方，那为何能成为目前最受欢迎的网络相册服务，并且被誉为最酷的 Web 2.0服务之一呢？
也许等介绍完了 Flickr 的所有特性你就会明白。

一、为用户着想

我们看到太多的网络相册服务，其界面让人眼花缭乱，到处是闪烁和浮动的广告，上传、浏览和处理相片的按钮被隐藏在一大堆用户不想看到的内容之间，有的操作步骤繁琐，有的不兼容非主流浏览器，总之生怕用户不点他们的广告或减少他们的页面浏览数，而不是为用户提供良好的体验。

Flickr 则像是当初界面称得上是简陋的 Google 一样，界面布局较为合理，配色以 logo 的蓝色作字体颜色，红色作偶尔的点缀，不追求花哨。除了登录后的界面之外很少看到明显的广告。现在国内的 Yupoo 和 Bababian 也是照搬它的界面风格。

其他诸如帮助文档、官方论坛、官方 blog 等等就不一一介绍了，反正只要英文过关，基本不会有什么为难的地方。

二、细节

脚本使用的地方很合理，如修改照片标题、把图片加入收藏或群组或图片集、批量操作、预览前后的图片、增加标签、下拉菜单等等；
除了 My Favorites 没有 RSS Feed 之外，其他我希望有 RSS 的地方它都有提供，很方便及时跟踪信息；
标签也几乎无处不在；
照顾了使用创作共用协议的用户；
无论是 Web 操作还是客户端都有提供，而且跨平台使用毫无问题；
对中文的支持很好；
各类信息的组织方式适合用户查看。

三、社会化功能

作为 Web 2.0 的带头羊，Flickr 的社会化功能是非常出色的，这个是介绍的重点：

Flickr Mail
Flickr 成员之间通信的基本工具，功能不算多，但是够用。

Contacts/联系人
包括一般的联系人、朋友和家庭成员3类，可以控制自己的图片只对哪部分联系人开放。登录后，可以看到联系人最新上传的部分相片。
添加联系人的地方比较隐蔽，要在对方的 Buddy Icon 上停留一下，再从下拉菜单中找，如下图所示：

Groups/群组

Fave](http://www.flickr.com/groups/bigfave/)，还可以是使用同一相机型号的同道。

群组中包括 Discuss、Map 和 Group Photo Pool。

Discuss 即群组讨论版块；
Map 是成员的地理位置图示；
Group Photo Pool 显示的是成员发送到群组的图片，在图片页面上点击“SEND TO GROUP ”按钮可以把图片加入你所属的某个群组中。这样 Group Photo Pool 就成为了群组成员公共的相册。

Tag/标签
给图片加上标签，如“中国”、“旅游”、“黄昏”等等。通过标签又可以发现同好或你感兴趣的图片。
Flickr 没有停留在简单的一级标签，它还通过 clusters 进一步细化标签，帮助你更快的找到需要的信息。例如 horse 这个标签下又细分了4组 cluster。

Badge/Slideshow
如果你拥有 blog，那么可以把自己的图片用 Flickr Badge 或 Slideshow 显示在 blog 上。

Favorites/收藏
看到喜爱的照片，就可以把它加入自己的收藏，并且共享给其他用户。虽然我的摄影水准令人不敢恭维，但是品位还是不算差的，所以我也以我的收藏为荣。
收藏还可以以 Flash 幻灯片形式展示出来。

Comments You’ve Made
跟踪自己的留言记录，免得忘记曾经到哪位的相册里面发表高论了。

用户在 Flickr 中能够拥有朋友、家人、同好、同乡等各种社会关系的圈子，也能够把许多由你创造和收集的信息方便快捷且美观地分享给其他人。

六、专注主题

曾经有一件有争议的事情影响比较大，就是一些用户因为上传太多屏幕截图而被 Flickr 暂时“封”了。原因是 Flickr 希望用户上传的都是自己拍摄的真实照片，如此才能够有效激发用户之间相互交流，他们不需要迎合那些仅仅把 Flickr 当作图标、按钮、横幅、截图图片存储站点的用户。

虽然后来 Flickr 方面有些松口，但是仍然警告用户，如果上传的图片中非照片的超过一半，很可能就会被暂时封掉，只有处理掉这些不合要求的内容之后才能够申请管理员重新审查并开通图片显示和上传功能。

对摄影爱好者来说，Flickr 的这个政策应该是好事情，起码图片的整体质量有基本的保证了。

七、注意事项

除了上面说的尽量少上传截图之外，粘贴图片时务必要选择 HTML 方式，即提供指向图片页面的链接。这样有利于读者访问你的 Flickr 页面并评论你的图片。

    • 　　转身慢、速度慢、没有默契的话，慎用造越位战术，何况有时候裁判的水平不高。
    • 　　又不是职业球员靠踢球混饭吃，没必要对一些判罚斤斤计较，老是和裁判过不去，当然，点球除外。对方有犯规越位，边跑边举手示意即可，千万不要自己停下来等裁判吹哨。
    • 　　当然，作为弱队，丢球是正常的，但丢球之后一定不能够陷于自责或互相埋怨，否则更多的丢球就会接踵而至。业余比赛中一旦某一方泄气了，一口气丢掉4、5个球都是可能的。所以最好有队长振奋士气，同时在前场利用犯规和出界拖延一下，给后场球员调整心态争取时间。
    • 　　确切地说就是要正确的认识自己球队的水平，不要抱有不切实际的幻想。这不是漫画，没那么多奇迹经常发生，根据球队的真实水平制定相应的战术才是王道。如果实力太差，丢球之后也要保持平衡的心态，否则容易演变成惨败或斗殴。

之所以写这一篇，是因为自己参加过的球队都属于这个档次的，有过不少惨痛经历，所谓久病成良医，看问题也看得比较清楚一点。由于是针对弱队的，自己又是防守球员出身，所以大部分是防守的要点。

　　1、首先要保证成员的基本功不至于太差，一定要有一定的底子和战术素养。

　　如果连符合这种要求的11个人都凑不齐，那么踢踢野球或只玩小场地就算了，否则被有经验的对手抓住某些位置的薄弱环节，一定死得惨。

　　2、如果有几个人选供选择，那么优先身体素质好的。

　　这不是青少年足球队搞未来的职业球员选拔，所以技术和意识上有天赋的球员不是球队重点培养的。既然是业余低水平球队，那么基本上有天赋也无出头之日，或者技术已经定型，没有上升空间。而在实战中，如果身体扛不住对手的冲击，空有技术和意识也无从发挥。

　　3、一定要保持阵形，并且注意补位。
　　业余球队最常见的问题就是有的关键位置如后腰、中后卫，往往头脑一热就向前猛攻，结果队友还没有到位球就给截了，退防、补防又不及时，本来完全可以守住的场面也会被对方抓住机会制造混乱甚至破门。
　　有的人可能会想，出其不意的助攻也许能够打乱对方的部署，创造得分机会。但在业余级别的比赛中，在场面没有优势的情况下，没有超群的个人能力而想要起到这样的作用，成功率是非常低的，相反，露出破绽被对方利用的可能性倒非常高。

　　如果防守的阵形不乱，中场的保护得当，后卫线的补位及时，那么即使遭到围攻，对方也难以得手，甚至会产生焦躁情绪。

　　4、注意卡位和盯人，不要只顾追着球跑，不要贸然上抢。
　　这又是一个菜鸟常犯的错误，类似红毛猴子的本能，区别在于，人家有漫画作家赋予的天赋，而你没有。
　　如果对方有精通带球和过人的角色，菜鸟的反应通常是3、4个人追着他跑，一进入肉搏距离就出脚抢球，然后被晃到失去重心，然后一个个被过掉，直到留下守门员面对单刀；如果对方有2、3个短传小配合打得出色的球员，菜鸟的表现通常是看到哪里有球就一股脑儿涌向哪里，然后对方一脚分球转移，后防立刻空虚。
　　比较有效的做法是，对方无球时可以稍微贴紧一点，不让他舒服地拿球转身，如果怕被瞬间加速抹过，也可以保持一点距离（针对力量型和速度型的对手应对方法不同，这个时候前面说的身体素质就很要紧了）。对方已经拿到球并且正面面向你的时候，应该随着他的推进慢慢后撤，等待其他协防人员到位。尽量把对方逼到外线，留心远射，也不要退得太远，不要让对方能够从容转移调动。如果对方选择人球分过，那么优先占住跑动路线而不是寄希望于一脚破坏，速度差距实在太大就阻挡犯规。
　　对方边前卫切入禁区了，不要全部跑向前点防守，注意防范禁区前沿和后点的插上进攻人员。

　　具体怎么做主要还是看个人的经验吧，自己和对手的身体、技术、反应等条件不同，应对的方案也不同。总的原则是不要被轻易突破，放边不放中。

　　5、后场处理球要果断。
　　边后卫切忌把球横传给禁区内的队友。一旦被断下，中后卫转身、补防不及，十有八九就是单刀。
　　控球分球不过关的，宁愿把球踢出边线、踢向前场也不要自己控球，特别是背对对方控球，那样很容易被截下。
　　守门员出击要果断，特别是用直塞球和过顶球打后卫身后的球，需要守门员快速出击。

　　守门员没有把握拿稳或击出禁区，就不要盲目摘高球，贴近横梁的大力远射托出去比较保险。

　　6、慎用造越位战术。
　　边后卫的位置一定要在中后卫之前，呈浅碟形站位，利于中后卫造越位。

　　转身慢、速度慢、没有默契的话，慎用造越位战术，何况有时候裁判的水平不高。

　　7、服从裁判。

　　又不是职业球员靠踢球混饭吃，没必要对一些判罚斤斤计较，老是和裁判过不去，当然，点球除外。对方有犯规越位，边跑边举手示意即可，千万不要自己停下来等裁判吹哨。

　　8、勇气和士气很重要。
　　水镜先生《士气论》有云︰战争，只是兵士间士气的一场竞逐…orz
　　在实力逊于对手的情况下，不惜命的奔跑和拼抢就是竞争的资本，如果连争个球也畏畏缩缩，那么这比赛不用打了。而士气的鼓动主要要靠队长、守门员和教练，队员自己也要有一股不服输的劲头。士气高昂的时候，遇到强大的对手也可能获胜。

　　当然，作为弱队，丢球是正常的，但丢球之后一定不能够陷于自责或互相埋怨，否则更多的丢球就会接踵而至。业余比赛中一旦某一方泄气了，一口气丢掉4、5个球都是可能的。所以最好有队长振奋士气，同时在前场利用犯规和出界拖延一下，给后场球员调整心态争取时间。

　　9、心态问题。

　　确切地说就是要正确的认识自己球队的水平，不要抱有不切实际的幻想。这不是漫画，没那么多奇迹经常发生，根据球队的真实水平制定相应的战术才是王道。如果实力太差，丢球之后也要保持平衡的心态，否则容易演变成惨败或斗殴。

　　10、重视定位球。
　　在控制力和传接球不如人的情况下，定位球往往是救命的法宝，一定要重视。
　　如果直接任意球开球者不够牛，那么最好不要直接射门，除非守门员容易脱手，一般情况还是往门前的位置旋去制造混乱比较可行。防守的时候则要注意盯人，争球时首先卡位扛人，解围一定要踢远，防止被外围轻松拿球瞄准了射门，这个时候守门员视线容易受阻。
