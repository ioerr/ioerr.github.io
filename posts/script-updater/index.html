Script Updater

在 Userscripts.org 安装了大量 Greasemonkey 脚本，却不能够像 Firefox 扩展那样自动监测新版本并安装，是不是让人觉得不好管理？

Script Updater 就是为了这个需求而出现的。安装之后自会列出一张已安装脚本的列表，如果脚本有了新版本，点击 Update 即可。但如果是 Userscripts.org 之外的站点上安装的脚本，就享受不到这个待遇了。
