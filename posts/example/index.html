Example

就像许多无聊的用户总会去尝试使用 username 作为用户名，password 作为密码，我也对如雷贯耳的 example.com 产生了兴趣。

与 www.com、helloworld.com 和 allyourbasearebelongtous.com 不同，传说中的 example.com

You have reached this web page by typing "example.com", "example.net", or "example.org" into your web browser.（这一句的潜台词就是：看吧，这全都是你的错，如果不是你蛋疼到输入这样的域名，也就不用来到这个毫无生趣的页面来。） These domain names are reserved for use in documentation and are not available for registration. See [RFC 2606](http://www.rfc-editor.org/rfc/rfc2606.txt), Section 3.（即是说，这些域名你们想都别想！有什么不满自己多学着点。）
