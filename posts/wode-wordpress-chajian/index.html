我的 WordPress 插件

2006-3-12 · calon

响应 Blogging Pro China
号召，列出我的
WordPress 插件，顺便庆祝 Weblogs.us 解封！

1、Adhensive
用于设置置顶帖。功能简单，但挺有用的。
由于不兼容2.x 版本，已经删除。

2、Akismet
防杀 Spam 的工具。
由于会影响 Weblogs.us 的稳定，因此被删除。

3、Better
Feed
增强 RSS Feed 内容。

4、Extended Live
Archives
可以实现 Live Archive 功能。

5、FancyTooltips
用 Javascripts 美化
Tooltip，把鼠标移到链接上就可以看到效果。可以手工添加，但插件形式要方便一点。

6、Random Quotes
如侧栏所示，用来随机显示摘录的言论。

7、SearchNow
即时显示搜索结果，见侧栏最上方。有时候会没有动作，需要多输入一个空格。

8、Subscribe To
Comments
留言者可以订阅回复。

9、Twilight
AutoSave
暂时保存在线输入的文章内容，防止丢失。
由于有良好的保存习惯，所以基本没用到，已经删除。

10、Ultimate Tag
Warrior
强大的 tag 工具。
由于觉得麻烦，所以删除。

11、WordPress Database Backup
用于数据库备份。

12、wp-Define
以 Tooltip 形式给指定的专有名词加上注释。
懒得维护，所以干脆删除。

13、WP Grins
在回复对话框前增加可点击的头像。

14、中文 WordPress 工具箱
显示最新回复、随机文章等等。

15、Dean’s Permalinks
Migration
无痛转换 Permalink 结构。

16、No Ping Wait
提高发布时 ping 的速度。

17、Spam Karma 2
由于 Akismet 无法使用，所以用这个来对付 spam。

18、SRG Clean Archives
产生一个简单的文章汇总页面。

19、Theme Switcher
快速切换主题。目前还有一点兼容性问题。

20、WordPress.com Stats
官方提供的简易访问统计工具。

21、Editing
Manager
为修改过的文章增加时间戳，同时可以控制是否在Feed中更新。
