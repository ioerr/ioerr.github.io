常用的工具软件

我常用的工具软件：

编辑器：
EmEditor，以前曾经多次介绍过这个功能强大、方便易用的文本编辑工具。支持正则表达式、丰富的搜索高亮显示、自定义选项多、人机界面设计出色、扩展能力强是它的优点。该软件需要付费使用，如果你是在校学生、研究机构，可以申请免费使用（成功率低），当然，国内的汉化版可以让你免费使用，不过不推荐。

UltraEdit，老牌文本编辑工具，被许多人看作是 Windows 下功能最强的编辑器。如果你是程序员，估计会经常使用它。虽然人机界面不如 EmEditor，但功能实用，足可独当一面。仍然要付费使用。 gVim，或许 Windows 下还有文本编辑器的功能超过 UE，但和 Vi 的衍生产品相比，绝对是小巫见大巫。虽然 Vim 面向 CLN 的人机界面让许多人望而却步，但 gVim 已经尽了很大的努力贴近普通的用户。如果经常使用 gVim，记住许多命令也不是太困难的事情吧？何况，这可是开源和免费的软件，比让你掏出美元或者违反用户许可协议要好吧。

播放器：
Media Player Classic，基本上不需要其他播放工具了。

Foobar 2000，但是播放音乐我还是选择这一位。

浏览器：
Firefox，结合丰富多样的扩展和主题，个人认为比 Maxthon 更顺手。当然也有不足之处，一些网页无法正常浏览，每个版本都有安全漏洞，尽管如此，综合评价还是最高的。

Opera，偶尔会使用它，所以有的好处没发现只能够怪我自己。

输入法：

智能狂拼2，用得最顺手的整句输入法，以前也多次介绍过。和一些程序有冲突，有的小地方也待完善，可惜3代太让人失望。 最新更新的智能狂拼3差强人意，整体来说仍然算是整句输入的首选。

搜狗拼音输入法，目前最好用的词组输入法。谷歌拼音的功能还差了点。

拼音加加2.204集合版，个人认为是最好的拼音词组输入法，集合版提供了实用工具，也不会修改首页设置了，虽然现在修改首页设置基本没什么用处。新一代同样让人失望。 下载工具：
BitComet，觉得和 BitSpirit 没有太多不同。近来很少用 BT 下载了…

使用 BT 下载的话，现在还是 uTorrent 最舒服啊~

eMule，目前最常用的下载工具，好处是不用上不兼容 Firefox 的 BTChina，能够找到一些很古老稀奇的资源。

Kugoo，稀缺资源较少，速度倒是快，偶尔有让人惊喜的内容出现。

[Net Transport]()，其实使用 Flashget 更多，但都用得很少了。 SoulSeek，英文 MP3下载的利器。

WinMX，MP3下载的利器，因为官司缠身，已经不提供下载，也不好找服务器了。 邮件客户端：
Thunderbird，主要在公司使用，替换了原来的 Foxmail。至于个人的邮件，当然都放在了 Gmail 中。

当要处理的邮件越来越多的时候，才知道还是 Becky! 好。界面虽然土了一点，但是胜在功能强、速度快。

Foxmail，需要使用远程管理功能时还是这个方便一点。好几次邮箱文件损坏，导致信件无法恢复，让我不敢用它保存重要邮件。 实时通讯软件：
Google Talk，这个出来以后我就废掉了 MSN Messenger，定位差不多，我更喜欢简洁一点的，人机界面设计更优秀的。在 GT 上的朋友不少，我也不是喜欢花很多时间单对单聊天的人，所以转移的成本也就小多了。

QQ/TM，宿命的遗忘、2Simple 等几个朋友多的群还在上面，所以必须保留 QQ。使用 TM 是因为它的界面稍微简洁一点。

阅读工具：
Foxit PDF Reader，比 Acrobat Reader 的启动速度和响应速度快多了，体积也很小巧。

ReadBook，偶尔还会用它看看书，不过能够让我花很长一段时间轻轻松松地坐在电脑前面看的 TXT/HTML 格式书籍不多了。

其他工具:
TypeAndRun，之前有过介绍，大大简化了我启动应用程序的操作。不过现在已经被后来者 Executor 取代了。

TrayIt!，用来把程序缩小至托盘区。 WinRoll，用于收起窗口只剩下标题栏。

[PPLive]()，看网络电视的工具，电视被抢占，球赛没转播的时候就靠它了。 UltraSurf，代理工具，有时候会无效。 翻墙，当然首选 TOR。

IME Tool，设置输入法的工具，关于输入法的疑难杂症可以找它。

PowerStrip，显示参数设置工具，在窗口模式下玩 CM 01/02就要用到。 EasyToon，制作黑白 GIF 图片的小工具。 Vcool，CPU 降温工具，我的老 Duron 在夏天有时是顶不住的。 i.Disk，用于分析硬盘上各个目录空间占用情况，方便整理空间。 Aqua Deskperience，截图软件，能够捕捉到鼠标选定不了了文本，如标题栏、目录树中的文字。缺点是按下截图热键后，画面不会静止不动，点击鼠标选定截图区域时，一些响应鼠标事件的对象会改变状态，因此菜单、托盘提示等地方的文本还是无法捕获。 ACDSee 2.44，古老版本的看图软件，够用就行。 FastStone MaxView 和 Irfan View 现在轮换着用来看图。

ePSXe/VBA/MAME/FinalBurn/Callus…，各种模拟器。

Daemon Tools，虚拟光驱软件，据说输入秘技可以让你凭空多出一个 CD-ROM。

NVU，网页编辑软件，至今还未用到顺手，不过本来就没有多少机会编辑大量网页… TextDiff，功能简单的文本比较工具，有时比 UE 的比较功能要好用，何况是开源、免费的。 Xplorer2 Lite，对个人免费的增强型资源浏览器，比有30天试用期的 Total Commander 好一点。而且其自带的文本编辑器还不错。 最终还是拜倒在 Total Commander 的脚下膜拜…

Dropbox 绝对是跨平台同步文件的首选。

TeamViewer 免费又小巧，还可以穿透内网，远程控制就靠它了。

照片管理和简单的处理交给了 Picasa。

以上的这些工具不包括工作时用的软件，其中有不少是使用了破解版，如果你知道有开源、免费、方便使用的工具，欢迎告诉我。
