Better Feed

2005-10-31 · calon

昨天试着安装了
BetterFeed
这个 WordPress 插件，现在在 RSS Feed
中应该已经可以看到效果了。如果你觉得有用，可以到这里下载并安装，装好之后到
Plugins - Plugin Editor 界面中按照自己的意愿修改或在本地修改好再上传。
我只在 RSS Feed
最下方增加了许多信息，没必要在多页浏览上搞什么花样，所以修改时在两个“FEEDFOOT”之间的位置加入自定义的文本即可。
我认为有用的变量：
* %%blogname%% : blog 的名称
* %%blogurl%% : blog 的地址（如 http://myblog.com/）
* %%posttitle%% : 文章的标题（如 Hello World）
* %%posturl%% : 文章的地址（如
http://myblog.com/archives/2001/02/03/hello-world/ 或者
http://myblog.com/?p=1337）
* %%id%% : 文章的 ID（如上面的1337，这在你的 Manager 界面中可以看到）
* %%date[Y]%% : 文章发布的日期时间，遵循 PHP date()
函数的语法格式，这在 Options 界面中可以定义
* %%categories%% : 文章所属的分类，用逗号分隔
* %%categorylinks%% : 文章所属的分类的链接，用逗号分隔
* %%comments%% : 评论的数目
* %%author_nick%% : 作者的昵称（如 Administrator、Calon）
* %%related[5]%% : 5篇相关的文章，数目可以自定义。需要有
* %%posturl%%#comments/ :
文章评论的地址，自动跳到第一条评论的起始位置
* %%posturl%%#respond/ : 文章最后输入回复的位置，自动跳到留言框
* %%posturl%%trackback/ : 文章 trackback 的地址
* %%posturl%%feed/ : 文章评论的 RSS Feed 地址

如果你有能力修改它的函数的话，相信可以在 RSS Feed 中加入更多的信息。需要注意的是，加入太多的内容也不太好，我增加这些信息只是为了方便那些无法直接访问的订阅者。
