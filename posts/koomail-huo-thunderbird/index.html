KooMail 或 ThunderBird

2006-1-16

今天把公司电脑上的 ThunderBird 用覆盖旧版本方式升级到了1.5，结果安装完毕之后竟然无法启动，重启之后依然没有解决问题。按某人的话来描述就是：

We uninstalled everything and reinstalled fresh. No go. We rebooted. No go. We uninstalled, rebooted and reinstalled. No go.
When one double clicks the icon, nothing happens. Task manager shows nothing. No windows. No event logs. No pop-ups. No sounds. No chicken nuggets. Nada.

用户体验可谓极差，安装之前也不打一声招呼。
于是临时找来 KooMail 作替代品。

KooMail 占用的资源不算太多，响应比较快，据说 DreamMail 的响应速度就很成问题，程序开得多的话，我的 ThunderBird 也经常反应迟钝。

远程管理的功能不错，可以在服务器上设置废件箱、草稿邮件箱、已发邮件箱。不知道其他客户端能不能够做到。

其他的功能嘛，和 FoxMail 和 DreamMail 都大同小异。自动分割大邮件、分析 IP 地址、建立虚拟子邮件夹的功能不错，但并不新鲜，而且无法建立多级虚拟子邮件夹，听说同时应用多个搜索条件也存在问题。

RSS 功能比较鸡肋，手工删除安装目录下的 RSS 目录或者在查看中禁掉即可。

过滤垃圾邮件的功能比 ThunderBird 差远了，和 FoxMail 比也不是一个档次。

看到作者挺乐于接受用户的建议的，于是去论坛留了一大段话，希望能够改进。

当然，后来在公司把 ThunderBird 卸载再重新安装，就没有问题了，实在是 *^!$%#^

有人用摩尔多瓦的 The Bat! 的么？没有特快专递，不支持 Yahoo! Mail、Hotmail，不支持新闻组，不支持一帐号多邮箱，不能够查看和编辑 HTML 源代码，不能显示外部链接图片…而且还不是免费的，除了 Educational License 之外，各版本价格在200-400元左右…好在现在的版本对 UTF-8 编码的支持要好些了…
在我看来就算打死也不用 Outlook，用 ThunderBird 也可以啊，不明白为什么许多人都说 The Bat!

Becky! Internet Mail 也是风评很好，但是一来又是收费的软件，二来那个界面实在太简陋了…

说来说去，目前还是 ThunderBird 最适合我，如果它占用的资源能够再少一点，功能再多一点－－说实话，它的插件真正实用的不多，而且更耗资源。
