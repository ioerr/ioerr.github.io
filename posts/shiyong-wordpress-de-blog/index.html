试用 WordPress.com 的 blog

2005-9-14 · calon

由于窝子的推荐，收到了
fdcn 发来的
WordPress.com 邀请，在此特别感谢两位！:D

这个 blog 的地址是http://calon.wordpress.com/

刚登入的时候无法更改初始密码，还以为有什么问题呢，结果过了10几分钟，修改密码顺利通过…看来确实是我的
RPWT。

之前就看过新的界面，还是更喜欢朴素一点的灰色界面。Dashboard
内容更加丰富了，虽然我通常只看右上角那一小块。

Write 界面最大的改变就是 WYSIWYG
的编辑器，如果不喜欢，可以在选项中关掉。

Manage 栏目中多了“邀请”一项。WordPress.com
的邀请只有一个，显得比较小气。一旦被邀请者申请 blog 成功，他的 blog
链接中会自动加上邀请者的 blog 地址。一大缺憾是没有 Files
子栏目，不能够编辑文件内容，这样就无法更改模板的设置了…相对应的，Presentation
下的 Theme 子栏目只能够选择不同的主题而已，没有 Theme Editor 子栏目。

Links
栏目中好像还没有提供链接导出功能，虽然自己动手没问题，但毕竟不如系统提供的功能方便。

Options 栏目中没有了 Plugins 子栏目！也无法更改上传文件的路径。看来
WordPress.com
是彻底断绝了使用插件的路子，不知道是不是因为 WordPress MU
是用于多人环境的，所以要安全第一？可不得不说，少了丰富的插件，WordPress
的魅力也就随之下降了很多。

Upload
里面的内容也没有多大变化。可以看到提供的上传空间有25M，传图片恐怕不够，而插件等等又没有用武之地…

虽然现在功能不够丰富，也许今后随着 WordPress MU
的升级，WordPress.com
会提供更多的功能吧，当然，最最要紧的，是不要立马被 GFW 给盯上了…

最后说一句最重要的，手头有一个邀请，不知道朋友中间哪一个真正需要，需要的请留言或发邮件到
calon.xu at gmail.com。优先宿遗的成员，不过我估计都用 MSN Spaces 或
Blogcn 去了，对 WordPress 感兴趣的不多。

    WordPress 软硬挨踢
