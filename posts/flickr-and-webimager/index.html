Flickr and WebImager

2006-2-27 · calon

开放 API
有什么好处？那就是有一堆人开发出各种各样的工具为你的服务留住用户。

虽然 Flickr
很不错，社区、评论、标签等功能挺有特色，但是相对其他图片存储服务优点也不是那么明显，例如速度、上传限制、浏览限制等等。好在还有其他的服务，如
Del.icio.us、FeedBurner
等等支持，以及这篇文章要介绍的 Flickr and
WebImager。

简单地说，这就是个 Flickr
专用上传工具。别看个头小，可是五脏俱全。功能包括：截图、打开指定的图片文件、上传、旋转、变换大小、调用画图板修改、增加鼠标指针图像。

操作说明见它自己提供的动画。

不看操作说明也很容易，首先点击那个 Check Flickr
的图标，到 Flickr 上开放 Flickr and
WebImager 通过 API
进行写的权限，然后经过一顿胡搞瞎搞胡搞瞎搞，记得点击 Upload 就行。

整个软件体积小巧、界面整洁、响应速度快、功能简单又贴心，Flickr
有了这样的工具，我只要担心每月的上传限额会不会很快用光了…

最后照例提几个缺点：
1、图标没有文字说明，小图标好歹有 tooltip
提示，截图用的大图标什么提示也没有，会让刚用上的用户犯晕。
2、不能够截取本身的图片…
3、如果上传之后能够返回该图片各种尺寸的地址供用户复制就好了，比打开
Flickr 再操作要方便一些。
4、如果有快捷键直接截屏就好了，否则一些下拉菜单、气球提示还是截不到…
5、确认和取消对话框中还有日文出现，怎么也得统一界面语言吧？
6、至少要有个 About 说明吧…
