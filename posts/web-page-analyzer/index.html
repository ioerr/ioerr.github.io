Web Page Analyzer

在海豚那里看到 Web Page Analyzer 这个站点工具，用来测试某个页面的载入速度，提交一份 Web Page Speed Report，告诉你对载入速度有影响的页面信息，以及改进的建议。

滞销书的首页只有 TOTAL_HTML 和 MULTIM_SIZE 两项测试是合格的，看来要多多改进了。对象和图片可能没有大幅削减的可能，Javascript 脚本倒是可以考虑考虑。
