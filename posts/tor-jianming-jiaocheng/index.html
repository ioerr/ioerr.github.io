TOR 简明教程

2006-3-6 · calon
Table of Contents
  • 1、下载：\

近日成功地拉拢我的表妹成为本 blog 的读者，因为要访问这个 blog 就需要突破
Great Fire Wall，所以教她学会了用 TOR。顺便放出简明教程。

1、下载：\#

的主页已经被屏蔽，所以没有办法直接下载到。可以到这里下载
TOR +
TorCP +
Privoxy 打包安装程序。感谢 Google
Pages 提供的服务！

如果今后有重大升级，我可能会更新下载地址。

2、安装：
没什么好说的，一步一步照着提示做就成了。

3、运行 TOR：
实际上是运行图形界面的 TorCP 和具备过滤内容、转换代理功能的 Privoxy。
从“开始”菜单-“所有程序”中找到 TorCP 和 Privoxy
并运行即可，也可以设置批处理命令或用 TypeAndRun 启动。
注意：打包安装版可能会设置 TOR 随电脑启动运行。

4、设置浏览器代理：
如果使用 IE，在“工具”-“Internet
选项”-“连接”-“局域网（LAN）设置”-“代理服务器”中，钩选“为 LAN
使用代理服务器”。
然后在“地址”一栏中填写表示本地 IP 的127.0.0.1，端口号设置为8118。

如果使用 MyIE/Maxthon，应该在工具栏上有切换代理的工具，老老实实地填写
IP、端口，启用即可。

如果是使用 Firefox，推荐使用
xyzproxy
插件。安装后在工具栏上
xyzproxy
图标的下拉菜单中选择“工具”-“编辑代理”，输入127.0.0.1:8118#Local。然后点击图标，使它由灰色变成彩色，便启用代理了。

5、其他说明：
理论上到上一步就能突破封锁了。可以用浏览器打开这个 blog，或者 WordPress
Documents、维基百科、新语丝、Google
的网页快照、BBC
中文网、中国网络审查条目等来测试是否成功。

如果通过 TOR 使用 WordPress 时出现和“Enable
Sending
Referrers”相关的错误信息，你就需要编辑
Privoxy 的 User Actions 文件，在{ fragile
}起始的一段中增加问题站点的地址，如 .weblogs.us
