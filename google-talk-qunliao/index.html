如何实现 Google Talk 群聊？

最难实现的是要有一台24小时在线的机器，如果没有的话下面不用看了，除非群组成员受得了。

要有一个专门用于作服务端的 Google Talk

帐号。

Python，体积不大，又很有用处，装一个不碍事。没有的话用低版本的可以不安装 Python。
最关键的工具 Conference Bot 在这里或者这里下载。

下面进入正题：

其实正题更短，因为运行 confbot.py 然后一步步照着填写就成。
运行的方法可以是在命令行下 python confbot.py，其实安装了 Python 的话，直接双击 py 文件也是一样的。
看到“Online!”的提示就表示成了。

详细说明见 这里，虽然版面有点乱…

补充一点点：
如果使用1.9.2版本，填写服务端帐号时邮箱地址只需要填@之前的部分，其他版本的情况不明。
如果填写有问题，删掉新生成的 ini 文件再执行 confbot.py 文件即可。
注意保护密码。

现在正在给里屋的 Google Talk 群作长期服务…

要查看当前活跃的群，可以去作者的页面。这就带来一个麻烦，因为你没办法经过审核以后再加人入群，所以可能会有不速之客闯进群里面。
