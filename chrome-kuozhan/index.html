Chrome 扩展
Table of contents

首先到这个页面下载 Chrome 4.0 Beta，然后去扩展页面淘吧。

Chrome 扩展的好处：
不需要重启就可以生效和卸载；
扩展有独立进程。

不佳之处则有：
官方扩展页面无分类；
浏览器内的扩展管理页面找不到扩展的主页；
使用其他浏览器时不能先下载再安装。

等到 Chrome 扩展的丰富程度超过 Firefox，而性能仍然能够保持最快的水准，我就会更换默认浏览器了。

目前我已经安装的扩展有：

Diigo bookmarks
书签收藏类，可惜 Diigo 至今仍然处于被封杀状态。

FlashBlock
阻挡 Flash 就可以节省很多资源了。

LastPass
密码管理。

Lazarus
自动保存输入框内容，防止丢失未提交的文本。

Session Manager
存储、还原会话。

Switchy!
多代理切换。

VerticalTabs
标签太多的时候可以以纵列形式查看标签标题文字。

不少 Firefox 的扩展要继续耐心等待才能在 Chrome 上看到，有的也许永远也看不到…
