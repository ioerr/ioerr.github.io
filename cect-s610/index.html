CECT S610
Table of contents

新手机到手，选择了性价比比较高，体积也没有一般的 PPC 那么吓人的 CECT S610（对应的 GSM 机型是 i-Mate 的 PDAL）。基本不需要玩什么游戏，也不指望拿来看电影、听音乐、拍照片，所以对硬件的要求不高；折腾得不多，所以不刷机也能用。
到手之后，除了传说中的电池续航能力是最大的问题之外，其他一切倒还好。

到手之后发现自己对 Windows Mobile 很不习惯，而且常用的应用不可或缺，于是立刻安装趁手的工具：

Total Commander for Pocket PC
据说 Resco Explorer 很好，但是对 TC 有感情，又是免费的版本，当然用 TC。

点讯输入法
自带的输入法始终不够用，而同时支持26键布局和双拼输入的点讯就成了首选，如果 A4有双拼方案才会再考虑。
点讯划词输入的创意不错，双拼方案也可以自己定义。

Haali Reader
电子书阅读软件选择太多了，好用、易用、免费、轻量级的随便挑一个就行。

UCWeb
Opera Mini 用久了换换口味，首选速度快、支持多窗口、免费的 UCWeb。

无限闹钟
自带的闹钟功能太弱，用功能强大、比较稳定的无限闹钟代替。好像作者已经停止了开发，但目前的版本已经足够用了，甚至多了许多不需要的功能。

快速菜单
既然没有手机上的 Executor，那就好好整理一下 Windows Mobile 的开始菜单，不要迷失在 \$tart 的汪洋大海中。

PockeTwit
Twitter 和 FriendFeed 上多位朋友推荐的 Twitter 手机客户端，感觉和网页版的要结合起来用才行。

ListPro
本质上只是一个清单管理工具，但以它为基础却可以实现丰富多彩的应用。
收费软件，分桌面和手机客户端，30天试用。

Jbed Java 模拟器
需要使用 Java 程序的时候才用得到，目前只装了个 Gmail for Mobile。
